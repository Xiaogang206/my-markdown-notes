**自动负性思维**（Automatic Negative Thoughts，简称 **ANTs**）是认知行为疗法（CBT）中最核心的概念之一，由Aaron Beck（认知疗法的创始人）在20世纪60年代提出，后来Daniel Amen等进一步推广并命名为“ANTs”。

### 简单定义
自动负性思维是指：在特定情境下，**大脑自动、快速、不经意识控制地冒出来的负性想法**。  
它们通常：
- **瞬间出现**（automatic），不需要刻意思考，就像大脑的“默认模式”
- **负向**（negative）：贬低自己、灾难化未来、过度解读他人
- **不经检验**：内容往往极端、片面、不符合现实或证据
- **情绪放大器**：这些想法一出现，就会迅速拉低情绪（焦虑、抑郁、自责、愤怒等），并驱动回避或自我破坏行为

一句话：**它们是大脑偷懒时给你的“负面滤镜”，让你把事情看得比实际更糟**。

### 为什么叫“自动”？
- 像条件反射一样快（几毫秒到几秒）
- 不需要你主动去想，它自己就跳出来了
- 很多人甚至意识不到它在后台运行，直到情绪已经低落或焦虑发作

### 常见表现形式 & 例子（2025-2026 CBT指南常用分类）
自动负性思维往往通过**认知扭曲**（cognitive distortions）表现出来。下面是15种最常见的类型（Beck原始列表 + Burns/David Burns扩展版 + Amen的ANTs常见归纳）：

| 类型（英文）                  | 中文常见译名          | 核心特征                              | 典型例子（日常场景）                                                                 | 它如何偷走好心情？ |
|-------------------------------|-----------------------|---------------------------------------|-------------------------------------------------------------------------------------|---------------------|
| All-or-Nothing Thinking      | 全有或全无 / 非黑即白 | 极端二分法，没有中间地带              | “考试没考满分，我就是彻头彻尾的失败者。”                                           | 一点小失误 → 彻底否定自己 |
| Overgeneralization           | 过度概括              | 用单一事件推及“永远”“所有人”         | “他没回我消息，我永远没人爱。”                                                     | 一次拒绝 → 终身孤独 |
| Mental Filter / Filtering    | 心理过滤 / 只见负面   | 只记住坏的，忽略所有好的              | 老板表扬了5句，只记住1句批评：“他觉得我一无是处。”                                 | 成功也被扭曲成失败 |
| Disqualifying the Positive   | 否定正面              | 把正面证据贬低、无效化                | “升职了？只是运气好/别人更差而已，我其实不行。”                                     | 成就感瞬间蒸发 |
| Jumping to Conclusions       | 妄下结论（读心+预言） | 没证据就脑补别人想法或未来灾难        | “她看我一眼就皱眉，肯定讨厌我。”<br>“面试肯定砸，明天就要失业。”                 | 无中生有制造焦虑 |
| Catastrophizing              | 灾难化                | 把小概率坏事想成世界末日              | “感冒了 → 肯定是重病 → 要死了。”                                                   | 小事瞬间变天崩 |
| Personalization              | 过度个人化            | 把无关的事当成针对自己                | “同事心情不好，肯定是我昨天说错话了。”                                             | 背负不该背的锅 |
| Should / Must Statements     | 应该/必须语句         | 用僵硬规则苛责自己/他人               | “我必须完美，不然就是废物。”<br>“他应该体谅我！”                                   | 制造内疚&愤怒 |
| Labeling                     | 贴标签                | 用单一词概括整个人                    | “我迟到了，我就是个没用的人。”                                                     | 自我攻击升级 |
| Magnification & Minimization | 放大负面 & 缩小正面   | 把缺点吹大、优点缩小                  | “犯了个小错 → 我彻底毁了。”<br>“别人夸我 → 只是客套。”                           | 失衡的自我评价 |
| Emotional Reasoning          | 情绪推理              | “我感觉是这样，所以一定是真的”       | “我感觉自己很失败，所以我真的失败了。”                                             | 情绪绑架现实 |
| Mind Reading                 | 读心术                | 确信知道别人在想什么（负面）          | “他没笑，肯定觉得我很蠢。”                                                         | 制造人际误会 |
| Fortune Telling              | 算命/预言灾难         | 预言负面未来如事实                    | “这次约会肯定黄，我注定单身。”                                                     | 提前绝望 |
| Always/Never Statements      | 永远/从不             | 极端绝对化语言                        | “我永远学不会这个。”                                                               | 剥夺改变可能性 |
| Blaming                      | 责怪他人/自己         | 把责任全推给某一方                    | “都是他害的我这么惨。” 或 “全是我不好。”                                           | 卡在怨恨或自责循环 |

### 如何快速识别自己中了哪些ANTs？
1. **情绪突然变差** → 停下来问：“刚才脑子里闪过什么念头？”
2. **念头一闪而过但很刺耳** → 很可能就是自动负性思维
3. **用“永远”“所有人”“一定”“必须”这类词** → 高度怀疑是扭曲
4. **写下来**：用“思维记录表”——情境 → 自动想法 → 情绪强度（0-100）→ 认知扭曲类型

### 为什么它这么危险？（预防抑郁角度）
- 这些想法像病毒，不断强化负性核心信念（“我不值得爱”“世界危险”“我无能”）
- 长期运行 → 情绪低谷 → 回避行为 → 生活质量下降 → 更容易触发抑郁发作
- CBT预防的核心就是**早早抓住它们、挑战它们**，别让“小蚂蚁”变成“大象”。

一句话总结：  
**自动负性思维就是大脑在“省电模式”下给你播的负面广播**——它快、狠、准，但往往是假新闻。学会“关掉广播”、换成更平衡的频道，就是CBT给你最实用的超能力。
