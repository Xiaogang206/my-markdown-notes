# 柏拉图

柏拉图（Plato）说的“画的鞋子不能穿，画的苹果不能吃”，并不是出自某个具体的寓言故事，而是他阐述其\*\*哲学核心“理念论”（Theory of Forms/Ideas）时，用来批评艺术（特指绘画和诗歌）\*\*的一个著名比喻。

这个比喻出自他的重要著作\*\*《理想国》（The Republic），体现了柏拉图对艺术的“摹仿论”\*\*的批判。

* * *

### 🏛️ 柏拉图的“理念论”和艺术的地位

要理解这句话，需要先了解柏拉图将世界划分为三个层次的观点：

#### 1\. 第一层：理念（Forms/Ideas）—— 真正的真实

**定义：** 这是永恒不变、绝对完美的本质。例如，真正的“美本身”、“正义本身”、“床的理念”。

**地位：** 这是世界的**真实原型**，是最高的实在。


#### 2\. 第二层：事物（Particular Objects）—— 摹仿理念

**定义：** 这是我们感官能接触到的、经验世界中的具体事物。例如，木匠制造的**一张具体的床**、农夫种出的**一个具体的苹果**。

**地位：** 它们是**理念的摹仿品**或影子，是不完善、会腐朽和变化的。


#### 3\. 第三层：艺术品（Art Objects）—— 摹仿的摹仿

**定义：** 艺术家的作品，例如**画家画的一张床**、**画的一只鞋子**。

**地位：** 它们是**对第二层事物的摹仿**。


### 🎨 批评艺术：“离真理三层远”

柏拉图认为，艺术品的致命缺陷就在于它是对真实的\*\*“摹仿的摹仿”\*\*，因此它：

|     |     |
| --- | --- |
| **批判点** | **含义** |
| **实用性为零** | **“画的鞋子不能穿，画的苹果不能吃。”** 这句话最直接的含义。艺术品只有表象，没有实体功能，无法满足人类的实际需求。 |
| **与真理隔绝** | 艺术模仿的是经验世界中具体事物（第二层）的**表面现象**，而非事物的本质（第一层理念）。因此，艺术是“离真理三层远”的。 |
| **可能具有欺骗性** | 艺术家往往只掌握了事物的**皮毛**，却能通过技巧使人产生错觉，以为自己看到了真实。柏拉图担心这会误导人们，阻碍他们去追求真正的知识（理念）。 |

### 简而言之：

柏拉图用“画的鞋子不能穿，画的苹果不能吃”来比喻：艺术品仅仅是幻象和影像，它看起来像真实的东西，但缺乏真实的本质和功能。因此，他认为艺术在理想城邦中应该受到严格的限制甚至驱逐，因为它无益于培养公民的智慧和道德。

# 亚里士多德

亚里士多德（Aristotle）虽然是柏拉图的学生，但他对老师的“理念论”和对艺术的批判提出了根本性的修正和反驳。

柏拉图认为艺术是“离真理三层远”的无用模仿，而亚里士多德则通过重新定义\*\*“摹仿”（mimesis）\*\*的性质和功能，极大地提升了艺术（特别是悲剧）的地位。

以下是亚里士多德反驳柏拉图“摹仿论”的主要观点：

* * *

### 1\. 重新定义摹仿：摹仿是人类的本能与学习途径

柏拉图将摹仿视为对低级现象的复制。亚里士多德则认为：

**摹仿是人类的本能：** 人类天生就喜欢摹仿。这是人类区别于动物的特征之一，也是我们学习知识、掌握技能的**最基本方式**。

**摹仿带来快乐和认知：** 我们从艺术品的摹仿中获得乐趣，因为我们能够通过观察艺术品**识别和学习**新的东西。即使是现实中令人厌恶的事物，通过艺术的摹仿也能带来认知上的快感。


### 2\. 摹仿的对象：摹仿“可能发生的”普遍性

这是亚里士多德反驳的核心。他认为艺术摹仿的不是具体事物的偶然性，而是事物内在的必然性、普遍性和可能性。

|     |     |     |
| --- | --- | --- |
| **对比** | **柏拉图的摹仿** | **亚里士多德的摹仿** |
| **对象** | 具体事物的**表象**<br>（“画的鞋子”） | 人的**行动、性格、情感**<br>（“像什么人”） |
| **性质** | **复制**<br>（对现象的复制） | **创造与重构**<br>（对可能性的揭示） |
| **功能** | 远离真理 | 揭示真理（普遍人性） |

亚里士多德认为，诗（文学）比历史更具有哲学性。因为历史只记载“已经发生”的个别事件，而诗（艺术）却描述“可能发生”的普遍真理。艺术描绘了在特定条件下，一个人会如何行动和遭受命运，这揭示了普遍的人性规律。

### 3\. 强调艺术的功能：**卡塔西斯（Catharsis，净化）**

柏拉图认为艺术会煽动情感、使人堕落。亚里士多德则提出了著名的\*\*“卡塔西斯”\*\*理论，认为艺术具有积极的社会和心理功能，尤其针对悲剧：

**定义：** 悲剧通过描写严肃而完整的行动，激发观众的**怜悯（pity）和恐惧（fear）等情绪，然后通过体验这些情绪，****使这些情绪得到净化、宣泄和缓解**。

**心理价值：** 卡塔西斯提供了一种**情绪上的平衡和健康**。它让人们在一个安全、虚拟的环境中体验并释放负面情感，从而在现实生活中更加理性、平静。


### 4\. 艺术具有自身的价值

柏拉图主要从伦理学和知识论的角度来贬低艺术。亚里士多德则明确了艺术的独立价值：

**艺术的自主性：** 艺术创作有其自身的规律和美学标准，不应完全服从于道德或政治的说教。

**技艺与完善：** 艺术本身是一种**技艺（_techne_）**，通过形式、结构和节奏的完善，能够实现其自身的**美学目标**。


总结来说：

亚里士多德通过将摹仿从\*\*“对低级现象的复制”提升为“对普遍人性和必然规律的创造性揭示”，彻底驳斥了柏拉图对艺术的谴责，并为西方美学奠定了坚实的基础。他将艺术视为一种重要的认知和情感体验\*\*，而非仅仅是无用的影像。
