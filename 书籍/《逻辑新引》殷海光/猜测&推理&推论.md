您提出了关于认知活动中三种知识获取方式的关键区别，逻辑学对此进行了精确的划分。根据您提供的材料，猜测、推理和推论在可靠性和思维的严谨性上存在明显的层级关系，并且在现代逻辑学中，推论被视为核心。

### 三者的可靠性排名

这三者都被视为间接的知识途径。按照所得知识的精细程度和可靠性排序如下：

|     |     |
| --- | --- |
| 方式  | 所得知识的可靠性 |
| **猜测** | 最粗、**最不可靠** |
| **推理** | 较精、**较可靠** |
| **推论** | 最精、**最可靠** |

### 具体的区别与定义

#### 1\. 猜测 (Guesswork)

猜测是知识获取方法中最无定轨或法则可循的一种。

* **依据：** 猜测多凭**天生的心智**、**直觉**，或是**经验的累积**。
* **过程：** 猜测的进行在意识界总是不大明显的，猜中的人**常常说不出一个所以然来**，他常靠**下意识作用**。
* **特点：** 猜测**总是不找根据的**，即使真有根据。因此，**猜测不是推论**。

#### 2\. 推理 (Reasoning)

推理（英文：reasoning）是一种基于特定原则的思维活动，它比猜测更加可靠。

* **依据（前提）：** 推理一定以**某一理为根据**。在推理时，是以“理”为前提的。
* **内容的特殊性：** 推理所依据的“理”是**分殊的**（特殊的）。例如，物理学之理与化学之理不同，因此各个推理有**不同的前提**和**不同的内容**。
* **范围：** 推理的范围是**非常之广**的。
* **逻辑地位：** 从前的逻辑家以为逻辑是推理之学，但依照现代逻辑眼光看来，**逻辑不是推理之学**。推理是推论的**周边**（peripheral parts）。

#### 3\. 推论 (Inference)

推论（英文：inference）是现代逻辑学的核心，它追求普遍的效准和必然的推论规律。

* **本质：** 推论是**必然有效的推论规律的科学**（即逻辑）。它是将一切推理中作为**特殊前提的“理”抽掉，所剩下的共同的“推”之程序**。
* **与推理的关系：** 推论是**一切推理所共同具有的中心程序**。逻辑所研究的不是一个个特殊的推理，而是为一切推理所必须依据的**推论形式**。
* **前提的纯粹性：** 推论本身是**没有而且不会有任何特殊的理作为前提的**。它不曾拿化学定律作前提，不曾拿物理定律作前提。推论形式中必须有\*\*“空位”\*\*（如变量），以便随时装进不同的“理”来行推演。
* **机制：** 推论必须借着**蕴涵关系**而行。推论的对或错**完全以推论法则为依据，不依靠经验或事实**。

### 总结三者的关系

推论是推理的中心程序，而推理是推论的周边。可以说，推理是在具体学科或情境中，运用其特定的“理”作为前提的推演活动；而推论则是形式化、抽象化的程序本身，即从这些特殊的“理”中抽象出普遍有效的逻辑形式。猜测则是一种缺乏明确理性和规律，主要依靠直觉和经验的认知活动。
