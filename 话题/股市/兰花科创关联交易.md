### 一、关联交易定义及主要类型

根据公司公告，兰花科创的关联交易主要包括以下类型：

2. **资产处置补偿**：如2025年6月对晋城市安达科工贸有限公司资产拆除的补偿协议；
3. **财务资助与担保**：2025年预计关联交易额22亿元，2026年预计规模相近。

# Gemini

兰花铁路公司（通常指“山西兰花铁路运输服务有限公司”）与兰花科创（山西兰花科技创业股份有限公司）的关系可以界定为：受同一控股股东控制的关联兄弟公司。

简单来说，它们是“同一个父亲”下的两个独立公司。以下是详细的关系梳理：

### 1\. 共同的控股股东：兰花集团

两家公司的核心纽带是山西兰花煤炭实业集团有限公司（简称“兰花集团”）：

**兰花科创：** 是兰花集团旗下的上市公司（A股代码：600123），也是集团的核心资产，负责煤炭、化肥、化工产品的生产。

**兰花铁路公司：** 全称为“山西兰花铁路运输服务有限公司”，是兰花集团的全资或控股子公司。


### 2\. 业务上的上下游协作

由于煤炭生产极度依赖铁路运输，这两家公司在业务上存在紧密的关联交易关系：

**物流承运：** 兰花铁路公司主要负责集团内部，特别是兰花科创各煤矿产区的铁路专用线维护、煤炭装发运输及物流服务。

**协同效应：** 兰花科创产出的煤炭，通过兰花铁路公司的运力运往全国各地。这种内部协作有助于兰花科创降低物流成本，并提高运输效率。


### 3\. 资本市场的最新变动（2025年动态）

值得注意的是，兰花科创近期在铁路和物流资产上有更深的布局，以增强其独立性：

**设立西北部铁路公司：** 2025年下半年，兰花科创与晋城市国投公司共同出资设立了“晋城市西北部铁路公司”，直接由上市公司参股，旨在打通其西北部矿区的运输瓶颈。

**收购港口资产：** 兰花科创还收购了“嘉祥港”股权，试图通过“铁水联运”将运输网络从单纯的铁路延伸到内河航运。

# 煤炭行业的关联交易

煤炭行业关联交易普遍存在主要源于三个结构性原因：

1. **产业链垂直整合需求**：煤炭企业通常控股运输、电力等下游公司，通过关联交易确保产销稳定性；
2. **资源禀赋地域性**：煤矿多位于资源型城市，当地配套企业常与煤企存在股权关联；
3. **政策导向影响**：保供任务下，国企集团通过关联交易协调内部产能分配。

但需注意：普遍性不等同于合理性，具体交易仍需分析定价公允性（可对比中国神华、中煤能源等同行数据）。

# Gemini

简单来说：关联交易是非常正常的商业现象，但在资本市场中，它是一把“双刃剑”。

法律并不禁止关联交易，甚至在很多行业（如能源、制造、地产）中，关联交易是由于历史原因或为了提高效率而必然存在的。

以下从三个维度为您拆解如何看待关联交易：

### 1\. 为什么说它“正常”？（正面意义）

对于像兰花科创这样的资源型企业，关联交易往往具有以下正面作用：

**降低交易成本：** 利用集团内部的铁路专用线（兰花铁路公司）或港口，沟通成本更低，运输保障更强。

**提高经营效率：** 内部供应渠道比市场招标更快、更稳定。

**资源共享：** 比如兰花科创向集团租赁土地或办公楼，可以避免大规模的一次性土地购买成本。


### 2\. 为什么监管要“盯死”它？（负面风险）

关联交易之所以敏感，是因为它容易成为大股东掏空上市公司的工具。主要风险点在于：

**利益输送（价格不公允）：** 比如上市公司高价买入关联方的劣质资产，或者低价把产品卖给关联方，导致利润被“合法”转移。

**资金占用：** 关联方以借款、担保等名义长期占用上市公司的现金，影响公司周转。

**独立性缺失：** 如果一家公司 50% 以上的生意都靠关联方（如兰花科创过度依赖兰花铁路），那么一旦关联方出问题，上市公司就会“失血”。


### 3\. 如何判断一笔关联交易是否“健康”？

作为投资者，您可以通过以下四个关键点来甄别：

|     |     |     |
| --- | --- | --- |
| **关注点** | **健康的表现** | **危险的信号** |
| **定价原则** | **公允价格：** 参考市场价，并在公告中明确对比数据。 | **价格异常：** 显著高于或低于行业平均水平。 |
| **决策程序** | **关联方回避：** 大股东代表和关联董事不参与投票。 | **强制通过：** 甚至不经过股东大会审议。 |
| **信披透明** | **详细说明：** 披露交易原因、背景、对业绩的具体影响。 | **语焉不详：** 只说“业务需要”，不解释必要性。 |
| **交易规模** | **占比稳定或下降：** 业务逐渐多元化，降低依赖度。 | **占比过高：** 离开关联方公司就无法独立生存。 |

### 以兰花科创为例

兰花科创在 2025年底 发布了关于 2026 年度日常关联交易的预计公告：

**规模：** 预计 2026 年日常关联交易总额约为 **33 亿元**。

**内容：** 主要是向关联方租赁土地、接受物流服务（如兰花铁路、云发工贸）等。

**观察点：** 兰花科创在公告中强调了“按原合同价续签”、“定价公允”。只要这些价格与第三方物流或周边地价一致，且经过了审计和独立董事签字，就是合规且正常的。


总结建议：

不要谈“关联交易”色变。只要价格公允、程序合法、信息透明，它就是提升效率的捷径；但如果一家公司突然发生大额、高溢价的关联资产收购，那就需要保持高度警惕了。
