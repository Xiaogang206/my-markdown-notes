在A股市场中，“300”开头的股票通常代表创业板（ChiNext），而\*\*“688”开头的股票代表科创板（STAR Market）\*\*。

虽然它们都属于“科技成长”阵营，但在定位、入门门槛和交易规则上有着显著的区别。

* * *

## 核心区别一览

|     |     |     |
| --- | --- | --- |
| **特性** | **创业板 (300)** | **科创板 (688)** |
| **所属板块** | 深交所 (SZSE) | 上交所 (SSE) |
| **板块定位** | 传统产业与新技术融合（三创四新） | 硬科技（半导体、生物医药、新材料） |
| **投资者门槛** | 10万资产 + 2年经验 | **50万资产** + 2年经验 |
| **打新单位** | 500股/手 | **500股/手** |
| **最小交易单位** | 100股起，1股递增 | **200股起**，1股递增 |
| **退市机制** | 相对严格 | 史上最严（无申辩期，直接退市） |

* * *

### 1\. 定位不同：应用 vs 核心

**300 创业板：** 更多是\*\*“老树发新芽”\*\*。它看重的是模式创新、服务创新。比如宁德时代、迈瑞医疗。它允许已经相对成熟、但仍具成长性的企业上市。

**688 科创板：** 必须是\*\*“硬科技”\*\*。针对的是国家战略新兴产业，很多企业处于研发投入高、甚至尚未盈利的阶段。比如中芯国际、寒武纪。


### 2\. 准入门槛：贫富分水岭

这是普通股民最直接的感受：

**创业板：** 权限开通要求前20个交易日日均资产不少于**10万元**。

**科创板：** 权限开通要求前20个交易日日均资产不少于**50万元**。这被称为“富人俱乐部”，旨在筛选出风险承受能力更强的投资者。


### 3\. 交易规则：细微的差别

虽然两者的涨跌幅限制都是 20%，且新股上市前5个交易日均不设涨跌幅限制，但细节上有坑：

**买入数量：** 创业板买入最小是100股；科创板买入最小必须是**200股**（之后可以1股递增，如201股）。

**价格笼子：** 两者都有2%的价格笼子限制，即买入申报价格不得超过基准价的102%，卖出不得低于98%，否则就是废单。


* * *

### 总结

简单来说，300创业板更接地气，很多细分行业的龙头都在这里；688科创板则更前沿、风险更高、门槛也更高，是纯粹的科技博弈场。
