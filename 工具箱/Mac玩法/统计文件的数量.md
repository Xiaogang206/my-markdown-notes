
**在 macOS 下，统计当前目录及子目录下所有 md 文件数量，最常用且最清晰的几种命令如下：**
该命令很成熟，可以直接用。

```
find . -type f -name "*.md" | wc -l
```



### 推荐写法（最常用）

```bash
# 方法1：最推荐，清晰直观
find . -type f -name "*.jpg" | wc -l
```

```bash
# 方法2：更精确（排除隐藏文件更彻底）
find . -type f -name "*.jpg" -not -path "*/\.*" | wc -l
```

### 各种常见写法对比

| 命令写法                                             | 是否包含子目录 | 是否统计隐藏文件/文件夹里的 | 说明              |                          |
| ------ | ------- | -------------- | --------------- | --- |
| `find . -type f -name "*.png"                    | wc -l`  | 是              | 是               | 最常用写法                    |
| `find . -type f -iname "*.PNG"                   | wc -l`  | 是              | 是               | 忽略大小写（png/PNG/PnG都算）     |
| `find . -type f -name "*.jpg" -not -path "*/\.*" | wc -l`  | 是              | 否（过滤掉 .开头的隐藏路径） | 常用于不想统计 .git/.DS_Store 等 |
| `find . -type f -name "*.mp4" 2>/dev/null        | wc -l`  | 是              | 是               | 忽略权限不足的报错信息              |
| `find ~/Downloads -type f -name "*.pdf"          | wc -l`  | 是（从指定路径开始）     | 是               | 从特定文件夹开始统计               |

### 实用例子

```bash
# 统计当前目录及子目录下所有 jpg 文件数量
find . -type f -name "*.jpg" | wc -l

# 统计照片文件夹里所有图片（不区分大小写）
find ~/Pictures -type f -iname "*.jpg" -o -iname "*.jpeg" -o -iname "*.png" | wc -l

# 只统计 .mp4 视频，不统计隐藏文件夹里的
find ~/Movies -type f -name "*.mp4" -not -path "*/\.*" | wc -l

# 统计当前项目里所有的 .ts .tsx 文件（前端项目常用）
find . -type f \( -name "*.ts" -o -name "*.tsx" \) -not -path "./node_modules/*" | wc -l
```

### 小技巧

- 想同时看文件列表 + 最后显示总数：

```bash
find . -type f -name "*.md" | tee /dev/tty | wc -l
```

- macOS 自带 `find` 和 `wc`，不需要安装任何东西。

**最简单记住的黄金写法：**

```bash
find . -type f -name "*.你要找的后缀" | wc -l
```
