普及的（distributed）是逻辑学中用来描述语句（特别是主宾词式语句，如 A、E、I、O 四种定言语句）中词端（term）的一种性质。

### 普及的定义和含义

* **定义：** 如果一个词端**所指涉的是一类的全部**，那么这个词端就是**普及的**。
* **非普及的（Undistributed）：** 如果词端所指涉的是**一类的一部分**或是**未定的部分**，那么这个词端就是未普及的。
* **重要性：** 词端普及与否的情况，与后续讨论的推论关系**至大**，因此必须弄清楚。在判断三段式推论是否有效时，词端的普及与否是一个**必须条件**。

### 四种语句类型中词端的普及情况

传统的逻辑学将四种主宾词式语句（A、E、I、O）的词端普及情况概括如下：

|     |     |     |     |     |     |
| --- | --- | --- | --- | --- | --- |
| 语句类型 | 名称  | 形式  | 主位词端 (S) | 宾位词端 (P) | 普及与否符号表示 |
| **A** | **全谓肯定语句** (All S is P) | 一切 S 是 P | **普及的** | **未普及的** | **○︶** |
| **E** | **全谓否定语句** (No S is P) | 没有 S 是 P | **普及的** | **普及的** | **○○** |
| **I** | **偏谓肯定语句** (Some S is P) | 有些 S 是 P | **未普及的** | **未普及的** | **︶︶** |
| **O** | **偏谓否定语句** (Some S is not P) | 有些 S 不是 P | **未普及的** | **普及的** | **︶○** |

详细解释：

1. **全谓肯定语句 (A) - 主位普及，宾位不普及（○︶）：**

* 例如：“凡属英雄都是好大喜功的”。
* 主位词端“英雄”指所有的英雄，或指英雄之类的一切分子，所以是**普及的**。
* 宾位词端“好大喜功的人”是**未普及的**，因为英雄只是好大喜功的人的一部分，二者发生关联的是该类的未定部分。
2. **全谓否定语句 (E) - 主位和宾位都普及（○○）：**

* 例如：“没有守财奴是慷慨好义的”。
* “守财奴”和“慷慨好义的人”都**普及的**，因为所有的守财奴都被排斥在慷慨好义的人之外，反之亦然，两个类的一切分子都涉及了排斥关系。
3. **偏谓肯定语句 (I) - 主位和宾位都不普及（︶︶）：**

* 例如：“有些思想家是性情孤僻的”。
* 主位词端“思想家”受“有些”限制，显然**没有普及**。
* 宾位词端“性情孤僻的人”也**没有普及**，因为“思想家”之类只与“性情孤僻的人”之类的未定部分发生联系。
4. **偏谓否定语句 (O) - 主位不普及，宾位普及（︶○）：**

* 例如：“有些诗人是不好饮酒的”。
* 主位词端“诗人”显然**未普及**。
* 宾位词端“不好饮酒的人”是**普及的**，因为“有些诗人”被排斥于**所有**“不好饮酒的人”之类以外。

### 普及性在推论中的应用

词端的普及性是检验某些推论（如三段式推论）有效性的关键规则之一。例如，在三段式推论中，有两条重要规律与普及性直接相关：

1. **中词必须普及：** 在两个前提之中，共词（中词）**至少必须普及一次**。如果中词一次都没有普及，则推论无效。
2. **潜越谬误：凡在前提中没有普及的词端，在结论中亦不得普及**。如果前提中未普及的词端（只指一类的一部分）在结论中潜越地指谓该类的全部（即变成普及的），则犯了潜越的错误，推论无效。
