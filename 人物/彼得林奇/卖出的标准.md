您的查询是关于彼得·林奇（Peter Lynch）的卖出标准，这是一个非常关键的问题，因为他认为卖出是一门艺术。林奇的卖出标准并非一个简单的公式，而是基于对公司基本面、股票类型和市场情绪的综合判断。

以下是彼得·林奇关于卖出股票的主要标准和信号：

### 一、 核心原则：关注公司基本面，而非价格波动

林奇强调，投资者的焦点不应该是表面的数字（如股价），而必须穿透表面，看到实际的公司，关注企业现在处于什么发展阶段，以及未来的前景如何。

1. **了解你为何买入：** 如果你清楚地知道当初买入一只股票的理由，那么自然就会清楚地知道何时应该卖出这只股票。
2. **避免执念：** 股票不会感谢你持有它，也不需要你对它有强烈的执念。投资者不应死抱住亏损的股票，想着“等到股价反弹到我买入价时才卖出”。这种执念可能会让资金的利用效率降低，无法将资金投入到更有潜力的公司上。
3. **不设武断的上限：** 林奇坚决反对设置人为的盈利目标（例如“股价翻一番就卖出”）。只要公司发展前景继续保持不变或变得更好，就应该坚决继续持有。
4. **警惕高估值陷阱：** 如果一只股票的价格被严重高估，即使公司其他情况都很好，也应该卖出。

### 二、 卖出信号：市场狂热与估值过高

当一只股票变得过于热门或估值达到不合理的水平时，林奇会考虑卖出，因为此时股价的上涨仅仅基于投资者一厢情愿的期望。

* **林奇的经典卖出标准（针对快速增长型股票）：** 如果有 **40 个**华尔街证券分析师都对一只快速增长型股票给予了最高的推荐评级；如果**机构投资者**已经持有这家公司 **60%** 的股票；如果**三种**国家级主流杂志都在极力吹捧公司的 **CEO**，那么这时绝对应该考虑卖出这支股票。
* **收益线与股价线的偏离：** 如果股价线远远**高过收益线**，或者**市盈率远远超过正常市盈率水平**，就应该考虑卖出这只股票，等待股价回落后再买回。
* **市盈率过高：** 任何一家成长股，如果以未来一年预期每股收益估算，**股价市盈率超过 40 倍**，就表明已经过于高估，此时买入十分危险。如果股票市盈率是收益增长率的**两倍**，那么这只股票亏钱的可能就非常大。
* **不切实际的预期：** 如果公司要维持其高市盈率所需的收益增长，必须达到**在商业上或物理上根本不可能实现的销售水平**（例如，雅芳需要销售十亿瓶香水，或者美国每两个家庭主妇中就有一个是雅芳的销售代表），这时应该卖出。

### 三、 针对六种类型股票的卖出时机

林奇建议根据公司所属的六种类型（缓慢增长型、稳定增长型、快速增长型、周期型、困境反转型、隐蔽资产型）来制定不同的卖出策略。

|     |     |
| --- | --- |
| 股票类型 | 卖出时机/信号 |
| **缓慢**<br>**增长型** | 股价上涨 **30%~50%** 时，或基本面恶化时。应警惕公司连续两年市场份额流失、削减研发费用、收购与主营业务无关的公司导致多元化恶化、或尽管股价低廉但股息收益率仍然很低的情况。 |
| **稳定**<br>**增长型** | 股价线超过收益线，或市盈率远远超过其历史正常水平。当公司增长开始放缓，并且通过削减成本来保持盈利的潜力有限时。 |
| **周期型** | 卖出**最佳时机**是在公司增长达到**顶峰周期即将结束**时。应关注以下信号：存货不断增加，且增加速度快于销售增长速度；大宗商品价格下跌（通常在公司收益出现问题前几个月发生）；公司决定用高额资本支出建立一家非常豪华的新厂。 |
| **快速**<br>**增长型** | **最重要的是不要过早卖出**。卖出信号包括：公司增长已接近**饱和期**（例如在每一个主要购物中心都开了店）；同一商店销售额降低；公司高级管理人员和核心员工辞职；市盈率相对于预期收益增长率而言高得离谱。 |
| **困境**<br>**反转型** | 卖出**最佳时机**是在公司成功**转危为安**之后。所有的困难都已经得到解决，并且每个投资者都知道这家公司已经东山再起了。此时应将该股票重新归类（例如，克莱斯勒从困境反转型转为周期型汽车公司）。 |
| **隐蔽**<br>**资产型** | 卖出**最佳时机**是在公司资产被**收购兼并者**发现并进行收购时。如果公司真的拥有大量隐蔽资产，投资者应该坚持持有，等待公司被收购。 |

林奇的卖出理念最终归结为：卖出基本面恶化并且股价上涨的股票，同时增持基本面好转并且股价下跌的股票。他的方法论是希望投资者能够像优秀的梭哈扑克玩家一样，在胜算大时增加赌注（买入），在局势不利时主动认输离场（卖出）。
