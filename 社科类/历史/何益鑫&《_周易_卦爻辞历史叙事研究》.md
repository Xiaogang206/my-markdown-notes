您提到的何益鑫所著《<周易>卦爻辞历史叙事研究》，其核心观点和研究主旨在于对《周易》卦爻辞的传统理解提出了根本性的颠覆和重新解释。

### 核心观点总结

作者认为，《周易》的卦爻辞原本是一部系统性的历史叙事，而非仅仅是抽象的哲学原理或简单的卜筮记录。

### 主要内容和论点

1. **《周易》的性质是“历史叙事”：**

该书主张，将《周易》视为占卜之书或纯粹的哲学义理之书，是后世的误读或发展。

卦爻辞的真正功用是记录和叙述一段连续的历史事件。

2. **叙事的时间跨度：**

这部历史叙事的时间范围是确定的，它涵盖了**商周之变的主体过程**。

叙事开始于**周文王即位之初**，经过**文王、武王**的时代，一直延续到**周公致政于成王**结束。

3. **对传统易学的批判与挑战：**

这一解释思路将《周易》的卦爻辞还原到具体的历史语境中，与传统的义理派（如强调阴阳、天道、哲理）和象数派（如强调数字、图形、占卜）的认识格格不入。

它提供了一种全新的、完备的解释原则，试图对《周易》六十四卦的所有卦爻辞都作出历史叙事层面的解读。


简而言之，这本书试图剥离《周易》后世被赋予的神秘、哲学色彩，将其定位为一部西周早期的史书，记录了周人如何推翻商朝、建立和巩固政权的历史进程。

# 义理派

义理派是中国古代研究《易经》的两个主要学派之一，与象数派相对。

义理派（或称“义理易学”）不侧重于《易经》的卦象、数字排列或占卜功能，而是将《易经》视为一部哲理书，致力于从卦爻辞和《易传》中发掘和阐发深奥的人生哲理、道德规范、政治原则和宇宙规律。

* * *

### 🏛️ 义理派的核心特点

|     |     |
| --- | --- |
| **特点** | **解释** |
| **重传轻经** | 侧重于研究《易传》（特别是《彖传》《象传》《文言》等），因为《易传》包含了对卦爻辞的哲学解释，是义理易学的主要思想来源。 |
| **注重义理** | 探求《易经》蕴含的**天道、人道、时位、变化**等抽象的哲学观念，如“自强不息”、“厚德载物”。 |
| **淡化卜筮** | 认为占卜只是一种外在形式，或只是启发哲理的手段。反对过分拘泥于卦象、数字和具体占卜方法。 |
| **目的** | 阐发治国修身、安身立命的道理，将《易经》转化为儒家思想的经典。 |

### 📜 义理派的代表人物

义理派的形成和发展贯穿了中国历史，其中最重要和最具影响力的代表人物是：

1. **王弼（魏晋）：**

**贡献：** 他是义理派的奠基人，著有《周易略例》。他主张\*\*“得意忘象”\*\*（领会了卦爻辞的哲理，就不必拘泥于具体的卦象），并提出了“贵无”思想，将玄学引入《易经》解释。

**批判：** 他的工作是对汉代繁琐的象数易学（如京房易）的**系统性批判和纠正**。

2. **程颐（北宋）：**

**贡献：** 宋代理学（义理易学的高峰）的代表人物之一，著有《伊川易传》。

**特点：** 他以**理学**（天理、人欲）来解释《易经》，强调《易经》是讲“理”的，把易学与儒家伦理道德紧密结合，是宋代义理易学的典范。


### 🔄 义理派与象数派的关系

义理派与象数派并非完全对立，而是相互补充，共同构成了中国易学的主流。

**象数派**侧重于**形而下**（具体事物、数字、时空、变化规律），试图通过数学和符号来预测吉凶。

**义理派**侧重于**形而上**（抽象哲学、道德伦理、人生智慧），试图通过文字和哲学来指导修身治世。


义理派的存在，使得《易经》从一部单纯的卜筮之书，上升为中国哲学史上重要的儒家经典之一。

# 象数派

象数派是中国古代研究《易经》的另一个主要学派，与义理派相对。

象数派（或称“象数易学”）侧重于《易经》的符号系统、图象、数字和结构，试图通过对这些“象”和“数”的分析，来推演宇宙万物的变化规律和预测人事吉凶。

* * *

### 🔢 象数派的核心特点

|     |     |
| --- | --- |
| **特点** | **解释** |
| **重经轻传** | 侧重于研究《周易》的**卦象、爻位、数字**等符号，认为卦象和数字本身就包含了宇宙的奥秘。 |
| **注重象数** | **象**指卦的形状、符号以及它们象征的事物（如乾为天、坤为地）；**数**指爻位、数字排列、阴阳数等。 |
| **强调卜筮** | 认为《易经》首先是一部占卜之书，其核心功能是预测未来、决断疑难，并指导人们趋吉避凶。 |
| **目的** | 通过数学和符号推演，揭示事物变化的规律，并服务于具体的预测活动。 |

### 📜 象数派的代表人物

象数派在中国易学史上源远流长，尤其在汉代达到了高峰，形成了庞大而复杂的体系。

1. **汉代易学家（象数派的鼎盛期）：**

**京房：** 创立了著名的“京房易”，是汉代象数派的集大成者。他将六十四卦配以日、月、星辰、十二律、六十甲子等，形成了一套极为繁复和精密的预测体系。

**郑玄：** 也是汉代重要的易学大师，通过对卦象和爻位关系的细致分析来解释《易经》。

2. **邵雍（北宋）：**

**贡献：** 宋代象数派的代表人物，他创立了著名的“**先天易学**”和\*\*“梅花易数”\*\*。

**特点：** 他以宇宙生成论为核心，用图式（如“伏羲六十四卦方圆图”）来推演宇宙万物的演化秩序和时间循环，将象数推向了高度抽象的数学和哲学层面。


### ⚔️ 象数派与义理派的辩证关系

**象数派**：强调\*\*“有象而后有辞”\*\*（先有符号和结构，后有文字解释），关注的是具体的“象”和“数”，以及它们之间的逻辑关系。

**义理派**：强调\*\*“得意忘象”\*\*（领会哲理，不必拘泥于符号），关注的是抽象的“理”和“道”，以及它们对人生的指导意义。


在宋代以后，理学家（如程颐、朱熹）试图融合象数与义理，朱熹的《周易本义》就吸收了邵雍的先天象数思想，并在注释中同时兼顾了象数和义理的解释。

总的来说，象数派为《易经》提供了独特的宇宙模型和预测方法，使《易经》在天文、历法、数学等领域产生了深远影响。
