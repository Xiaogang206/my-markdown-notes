**Cowork**（全称通常写作 **Claude Cowork**）是 Anthropic 公司在 **2026年1月** 正式推出的一个 **AI 代理功能**，目前属于研究预览（research preview）阶段。

简单来说，它让 Claude（也就是我背后的那个大模型）从“只会聊天回答问题”升级成了“能像同事一样自己干活”的模式。

### Cowork 的核心特点（和普通聊天模式最大的区别）

| 方面   | 普通 Claude 聊天模式 | Cowork 模式                     |
| ---- | -------------- | ----------------------------- |
| 交互方式 | 你问一句、我答一句      | 你交代一个目标，它自己拆解+执行              |
| 文件处理 | 只能处理你手动上传的内容   | 可直接读写你授权的本地文件夹里的文件            |
| 执行能力 | 基本只能输出文字、代码    | 能创建/修改文件、整理资料、调用浏览器           |
| 自主性  | 基本被动等待指令       | 主动规划步骤、多步执行、甚至并行处理            |
| 使用场景 | 问问题、头脑风暴、写点东西  | 真正“上班”：整理发票、做数据分析、写报告、整理项目文件等 |
| 需要条件 | 浏览器或App就能用     | 需要 Claude Desktop 应用 + 授权文件夹  |

官方 slogan 是：  
**“Cowork: Claude Code for the rest of your work”**  
意思就是：Claude Code 是给程序员写代码用的，Cowork 是给剩下所有人的“非编程工作”准备的。

### 典型使用例子（2026年真实用户场景）

- “把这个月所有发票按日期+金额整理成 Excel 表格”
- “分析我电脑里这个项目文件夹的所有周报，给我写一份季度总结 PPT”
- “去把小红书上竞品笔记抓下来，按品类分类整理”
- “帮我把桌面这个乱七八糟的文件夹按类型归档”

它会先给你规划步骤 → 然后自己去读文件 → 处理 → 写新文件 → 最后告诉你完成了。

### 当前（2026年2月）使用门槛

- 需要 macOS 或 Windows 的 Claude Desktop 应用
- 需要订阅付费计划（Pro / Max / Team / Enterprise），早期只限 Max，后来下放到 Pro
- 首次使用要授权特定文件夹访问权限（很重要：它能改你文件，所以要给对的文件夹）

一句话总结：

**Cowork = Claude 的“数字员工”模式**  
不再是问答机器人，而是能真正帮你在电脑上“干活”的 AI 同事。

如果你最近看到有人说“Claude Cowork 太强了”“AI 终于开始上班了”，指的就是这个。
