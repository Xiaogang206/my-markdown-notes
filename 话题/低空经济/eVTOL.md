eVTOL
eVTOL 的全称是 Electric Vertical Take-off and Landing，中文直译为\*\*“电动垂直起降飞行器”\*\*。

简单来说，你可以把它理解为\*\*“放大版的载人无人机”或者“电动化的飞行的士”\*\*。它是未来“城市空中交通”（UAM）场景下的核心载体。

* * *

### 一、 核心定义：拆解 eVTOL 的三个关键词

1. **Electric（电动）**：

**能源**：完全或主要依靠电池驱动电机（类似电动汽车）。

**优势**：零排放、低噪音（仅为直升机的 1/10 甚至更低）、结构简单（维护成本比燃油发动机低得多）。

2. **Vertical Take-off and Landing（垂直起降）**：

**无需跑道**：它不需要像普通飞机那样滑跑，可以直接在楼顶、草地或专用起降坪垂直起飞和降落。

**灵活度高**：非常适合高楼林立、地面交通拥堵的城市环境。

3. **Aircraft（飞行器）**：

它不是普通的玩具无人机，而是具备严格航空标准、可以载人（通常 2-6 人）或载重物的新型飞行器。


* * *

### 二、 eVTOL vs 传统直升机：有什么区别？

虽然它们都能垂直起降，但底层逻辑完全不同：

|     |     |     |
| --- | --- | --- |
| 特性  | 传统直升机 | eVTOL |
| **动力源** | 燃油（涡轮轴发动机） | **电力（电池+电机）** |
| **噪音** | 巨大（像施工现场） | **极小（像大风扇扫过）** |
| **安全性** | 依赖单/双主旋翼（一旦故障风险大） | **分布式动力（DEP）**：拥有 8-16 个旋翼，坏掉两个也能平稳降落。 |
| **驾驶** | 需要专业飞行员，操控极其复杂 | **高度自动驾驶**：未来目标是“一键飞行”，无需飞行员。 |
| **成本** | 极其昂贵（千万级购买，万级运行） | **相对低廉**：目标是让普通人也能像打车一样“打飞的”。 |

### 三、 常见的三种“长相”（技术路线）

1. **多旋翼型 (Multicopter)**：

样子就像大号的大疆无人机。结构最简单，但飞不快、飞不远。适合同城短途（如：亿航 EH216-S）。

2. **复合翼型 (Lift + Cruise)**：

有专门负责垂直起跳的旋翼，还有专门负责向前飞的推进桨和机翼。起飞后像飞机一样滑翔，效率更高（如：峰飞、沃兰特）。

3. **倾转旋翼型 (Tiltrotor)**：

旋翼可以“点头”。起飞时向上，飞起来后转向前。技术最难，但速度最快、航程最远（如：Joby Aviation、沃飞长空）。


* * *

### 四、 为什么要搞 eVTOL？

目前的城市交通正面临“地面极限”。eVTOL 的出现是为了把城市通勤从“2D 平面”提升到“3D 空间”：

**医疗救护**：救护车堵在路上时，eVTOL 可以实现“分钟级”跨院转运。

**商务通勤**：从市中心到机场，开车 1 小时，eVTOL 仅需 10 分钟。

**旅游观光**：提供低空视角的城市或景区导览。
