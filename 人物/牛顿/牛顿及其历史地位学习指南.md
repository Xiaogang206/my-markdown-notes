# 牛顿及其历史地位学习指南

## 测验题

说明： 请根据源材料内容，用2-3句话简要回答以下十个问题。

1. 牛顿在数学和物理学领域的主要贡献是什么？
2. 为什么说牛顿最了不起的成就并非发现知识点，而是构建了一个“三维的立体的学科体系”？
3. 哈雷彗星的发现如何证明了牛顿理论的巨大影响力？
4. 除了科学成就，牛顿晚年还担任过哪些重要职务？这些职务如何体现了他在英国社会中的地位？
5. 牛顿去世后获得了什么罕见的荣誉？为什么说这一荣誉的授予过程非常顺利？
6. 请描述牛顿葬礼的盛况及其在威斯敏斯特大教堂的墓地。
7. 哲学家伏尔泰与牛顿有何关联？牛顿的葬礼对他产生了什么影响？
8. 关于牛顿被苹果砸中而发现万有引力的著名故事，其起源是什么？这个故事的真实性如何？
9. 视频演讲者为何认为科学家比政治家更伟大？请阐述其主要论点。
10. 牛顿在担任皇家造币厂厂长期间，为解决当时硬币“剪边”问题做出了什么具体贡献？

* * *

## 测验题答案

1. 在数学上，牛顿发现了二项式定理并与莱布尼茨各自独立发明了微积分，从而开创了高等数学。在物理学上，他提出了力学三大定律和万有引力定律，为经典力学奠定了基础，并在光学和天文学领域做出了开创性贡献。
2. 因为在牛顿之前，物理和数学知识是零星的、不成体系的经验总结。牛顿则建立了系统性的学科，清晰地定义了质量、力、惯性、速度和加速度等基本概念，使自然现象变得可被解释和预测。
3. 牛顿的同代科学家哈雷利用牛顿的理论，准确地预测了一颗彗星的回归时间。73年后，这颗彗星如期回归，并被命名为哈雷彗星，这有力地证明了牛顿理论体系的预测能力和正确性。
4. 牛顿晚年曾担任英国皇家学会会长长达24年，还担任过皇家造币厂厂长。这些职务表明他不仅是杰出的科学家，还在英国的政治、经济领域拥有极高的地位和影响力。
5. 牛顿去世后获得了国葬待遇，这是非王室成员极少能享有的荣誉。由于他在科技、政治、经济和宗教等多个领域都有卓越成就且门生故旧遍天下，英国议会在讨论时几乎没有争议，一致批准了该待遇。
6. 牛顿的葬礼极其隆重，英国及欧洲各国的王室贵族和名流均有参加，并由一位大法官、两位公爵和三位伯爵为他抬棺。他被安葬在威斯敏斯特大教堂，其墓地位置比许多国王的还醒目，并配有纪念其科学功绩的精美雕像。
7. 法国哲学家伏尔泰流亡英国时恰好参加了牛顿的葬礼，并为英国人对科学家的尊重所震撼。这激发了他研究牛顿的兴趣，使他成为牛顿的“铁粉”，牛顿的思想也对他自己的哲学体系产生了影响。
8. 这个故事相传起源于伏尔泰。他向牛顿的一位亲戚询问万有引力的发现过程，对方不耐烦地随口说是被苹果砸了脑袋，伏尔泰将此说法完善并流传开来。牛顿本人的任何手稿或书面记录中都没有这一记载，因此被认为是历史误会之一。
9. 演讲者认为，政治家的影响力受限于特定国家和时代，而牛顿等科学家发现的法则是普适和永恒的，适用于从宇宙诞生到结束的整个宏观低速运动。真正的伟人无需权力和武力让人服从，因为他们揭示的自然法则本身就是必须遵守的。
10. 为了防止不法分子剪掉贵金属硬币的边缘（剪边）以牟利，牛顿在担任厂长期间发明了在硬币边缘设计一圈锯齿。这种设计使得任何剪边的行为都极易被发现，从而有效遏制了这种损害货币价值的犯罪行为。

## 论述题建议

说明： 以下为五道论述题，旨在启发深入思考，不提供标准答案。

1. 结合视频中提到的牛顿在数学、物理学和天文学上的贡献，论述为什么他被认为是构建了“一个三维的立体的学科体系”，而不仅仅是发现了一些知识点。
2. 分析牛顿获得国葬待遇的历史背景和社会原因。为什么一位科学家能在当时的英国社会获得与君王将相媲美的身后荣誉？
3. 探讨伏尔泰参加牛顿葬礼这一事件对启蒙运动思想的潜在影响。英国社会对科学家的尊重如何通过伏尔泰传递并影响了欧洲大陆？
4. 视频演讲者将科学家与政治家进行对比，并断言前者更为“伟大”。请评述这一观点，并结合牛顿的例子以及你自己的理解进行论证。
5. 从牛顿担任皇家造币厂厂长的经历出发，讨论杰出科学家在社会公共服务领域可能扮演的角色和发挥的作用。

## 关键术语词汇表

|     |     |
| --- | --- |
| 术语  | 定义  |
| **国葬 (Guózàng)** | 一种为对国家有杰出贡献的人物举行的隆重葬礼仪式，通常需要经过议会批准。牛顿是英国历史上第二位享受此待遇的非皇室成员。 |
| **威斯敏斯特大教堂 (Westminster Abbey)** | 位于伦敦的一座著名教堂，是许多英国君主和伟人的安葬之地。牛顿的墓地位于其中一个非常醒目的位置，周围还安葬着霍金和莎士比亚等伟人。 |
| **皇家学会 (Royal Society)** | 英国著名的科学机构。牛顿曾长期担任其会长达24年，其葬礼抬棺人也均为皇家学会会员。 |
| **皇家造币厂 (Royal Mint)** | 负责制造英国硬币的机构。牛顿曾担任其厂长，并在此期间进行了重要改革，如为硬币设计锯齿边缘以防“剪边”。 |
| **伏尔泰 (Voltaire)** | 法国启蒙运动之父，一位著名的思想家。他因参加牛顿的葬礼而深受震撼，成为牛顿的追随者，并传播了关于牛顿与苹果的故事。 |
| **哈雷 (Halley)** | 与牛顿同时代的科学家。他利用牛顿的理论成功预测了哈雷彗星的回归，从而验证了牛顿力学体系的强大。 |
| **亚历山大·蒲柏 (Alexander Pope)** | 18世纪最伟大的古典诗人之一。他为牛顿写下了一句著名的墓志铭：“自然与自然的法则隐藏在暗夜中；上帝说，让牛顿去吧！于是一切豁然开朗。” |
| **《大国崛起》 (The Rise of Great Powers)** | 2006年由中国中央电视台（CCTV）推出的一部纪录片，讲述了九个世界强国的崛起过程。该片在讲述英国的部分，花费了大量篇幅介绍牛顿。 |
| **金本位制 (Gold Standard)** | 视频中提及牛顿在皇家造币厂任职期间，用黄金代替白银与英镑挂钩，这是金本位制的早期实践。 |
| **剪边 (Clipping)** | 一种不法行为，指剪掉由贵金属（如白银）制成的硬币边缘以获取金属，导致流通中的硬币实际价值受损。牛顿通过在硬币上增加锯齿来防止此行为。 |
