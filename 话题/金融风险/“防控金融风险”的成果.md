从2017年第五次全国金融工作会议将“防控金融风险”列为金融工作三大任务之一、党的十九大将“防范化解重大风险”作为三大攻坚战之首（重点防控金融风险）以来，中国在防止发生系统性金融风险方面取得了显著成果。防范化解金融风险攻坚战取得**重要阶段性成果**乃至**实质性突破**，金融风险整体收敛、总体可控，金融体系经受住了疫情冲击、外部环境复杂变化等多重考验，牢牢守住了不发生系统性金融风险的底线。**中国成为近几十年极少数未发生系统性金融危机的主要经济体之一**。

以下从关键维度总结2017年至今的主要成果（基于中央权威文件、监管部门报告及最新进展，至2025-2026年数据）：

### 1. **总体成效与宏观层面**
- **金融杠杆率与脱实向虚得到根本扭转**：2017年以来，银行业总资产增速大幅放缓（年均约8%，不及此前一半），影子银行规模较历史峰值压降**超25-30万亿元**，同业理财、委托贷款、信托贷款等高风险业务大幅收缩。宏观杠杆率从快速上升转为先稳后升，但整体可控，企业部门杠杆率稳中有降。
- **不良资产处置力度空前**：2017-2021年主动处置银行业不良资产超**12万亿元**，后续持续推进，2022年后累计处置规模进一步扩大。不良贷款认定更严格，处置更市场化。
- **高风险机构数量与规模大幅压降**：高风险中小金融机构（尤其是中小银行）数量和资产占比较峰值显著下降，部分省份实现“动态清零”。风险完全可控，外资机构报告也认可中国高风险金融资产规模大幅下降并将继续缩减。
- **系统性风险防控能力显著增强**：建立健全早期纠正机制、宏观审慎+微观监管双支柱框架、金融稳定保障基金初步筹集、金融稳定法立法推进。经受住疫情、国际金融动荡等考验，未出现系统性危机。

### 2. **重点领域风险化解成果（三大重点领域）**
这些是中央反复强调的传染源头，已取得实质进展：

| 重点领域                 | 主要风险点                  | 2017年以来主要举措与成果（截至2025-2026年最新进展）                                                                                                                                                                       |
| -------------------- | ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **地方政府债务风险**（隐性债务为主） | 融资平台债务规模大、土地收入下降导致偿债压力 | - 建立“一揽子化债方案”，债务置换、重组、展期、借新还旧。<br>- 2023-2025年多轮政策：严肃财经纪律、剥离政府融资功能、转型市场化主体。<br>- 最新：2024-2025年使用结存限额超5000亿元化债；融资平台数量下降超60%，金融债务规模下降超50%；风险水平大幅收敛，增量基本控制，存量有序化解。                                        |
| **房地产金融风险**          | 房企高杠杆、“三道红线”、房价调整、交付风险 | - “房住不炒”+因城施策：优化首付比、利率下限、存量房贷利率下调（惠及超5000万户家庭，年减支约3000亿元）。<br>- 保交楼：城市房地产融资协调机制“白名单”项目贷款超7万亿元，支持近2000万套住房交付。<br>- 三大工程（保障房、城中村改造、平急两用）资金支持超1.6万亿元。<br>- 最新：推动“好房子”建设、构建新发展模式，房地产金融化泡沫化势头实质扭转，风险外溢得到控制。 |
| **中小金融机构风险**         | 治理不规范、不良率高、区域性传染       | - “一省一策”改革化险：兼并重组、在线修复、市场退出。<br>- 高风险机构数量/规模较峰值大幅压降，风险占比低。<br>- 最新：农信社改革超半数省份完成省级法人组建；城商行、股份制银行高质量发展推进；高风险中小银行数量明显减少，部分省份“动态清零”。                                                                       |

### 3. **制度与监管层面成果**
- **监管体制重大升级**：2017年成立国务院金融稳定发展委员会；2018年银保监会合并；2023年组建中央金融委员会、中央金融工作委员会+“一行一局一会”格局，强化党中央集中统一领导、穿透式监管、功能监管、行为监管。
- **长效机制初步构建**：金融稳定法推进、金融稳定保障基金筹集、早期纠正硬约束机制、非法金融活动常态化打击（P2P全部清零、互联网金融转入常态监管）。
- **服务实体与风险防控并重**：金融服务实体经济质效提升（制造业、科技、绿色、普惠贷款高速增长），同时严防风险反弹。

总体而言，2017年以来，中国通过“稳定大局、统筹协调、分类施策、精准拆弹”的方针，成功将高风险从“水落石出”转为“收敛可控”，为高质量发展和金融强国建设筑牢了安全底线。风险虽仍存（外部冲击、结构性问题），但防控能力已大幅增强，未来将继续坚持“早识别、早预警、早暴露、早处置”，统筹发展和安全。
