“道德是自律而非他律”这一观点是伦理学中的一个重要立场，尤其与\*\*康德（Immanuel Kant）\*\*的道义论哲学紧密相关。

它的核心在于强调**道德行为的价值来源于主体自身的意志和内在动机**，而不是外部的压力、规则或奖惩。

* * *

### 1\. 道德的“自律”是什么？

\*\*自律（Autonomy）\*\*是指道德主体（人）**自己给自己立法**，自觉地、自由地、心甘情愿地遵循道德原则。

**动机来自内在：** 行为不是出于害怕惩罚或渴望奖励，而是仅仅因为**认识到并尊重该行为的道德正确性**。

**主体是立法者：** 你遵循的道德律是你自己作为理性人所认可和接受的普遍法则。你不是被动地接受命令，而是主动地选择服从理性。

**体现自由意志：** 只有出于自律的行为才真正体现了人的自由和尊严，因此，这种行为才具有真正的道德价值。


### 2\. 道德的“他律”是什么？

**他律（Heteronomy）是指道德主体被****外部力量所约束和驱动**，他的行为遵循外部的规则或动机。

**动机来自外部：** 行为不是为了道德本身，而是为了达到**外部目的**，例如：

**害怕惩罚：** 怕被警察抓住、怕被社会指责。

**追求利益：** 想要获得赞扬、奖励或天堂的福报。

**服从命令：** 盲目服从权威的指示。

**主体是被动者：** 行为的驱动力是**非理性的**（如恐惧、欲望），人只是实现外部目的的工具。

**缺乏道德价值：** 在康德看来，出于他律的行为，无论后果多么好，都只是“合乎道德”（外在符合），而非“道德的”（内在动机），因为它缺乏真正的道德价值。


### 3\. 为什么说道德“是自律而非他律”？

如果道德取决于他律，那么它将面临以下困境，从而失去其稳定性、普遍性和至高性：

|     |     |
| --- | --- |
| **困境** | **解释** |
| **普遍性**<br>**丧失** | 如果道德由文化、宗教或权威决定，那么道德律就会**因地而异、因人而异**，无法成为所有理性人都应遵循的普遍法则。 |
| **稳定性**<br>**不足** | 如果道德取决于外部的奖惩或利益，那么一旦环境变化，**利益动机消失**，人就会放弃道德行为。 |
| **价值的**<br>**贬低** | 如果我行善是为了得到回报，那么行善就成了一种**交易**，而不是一种内在的义务或尊严的体现，从而**贬低了道德的价值**。 |
| **剥夺**<br>**自由** | 只有自律，即基于理性对普遍法则的自觉服从，才能被称为真正的**自由**。他律下的服从，本质上是受制于欲望或恐惧，反而是一种不自由。 |

因此，这一观点认为，只有当人出于**对道德法则本身的敬重**，并**自由地选择**遵循它时，他的行为才是真正**道德的**。这就是说，道德的本质在于**自律**。
