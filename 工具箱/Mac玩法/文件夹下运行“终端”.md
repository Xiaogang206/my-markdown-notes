在 Mac 上，有几种非常便捷的方法可以直接在指定文件夹“原地”开启终端，省去了手动输入 `cd` 路径的麻烦。

### 结论
1. 选中文件夹，快捷键 **`Control + Option + T`** 
2. 文件夹右键，服务，新建终端窗口
3. 拖拽，摁住cmd拖拽可以不用输入cd
4. 路径导航栏

### 方法一：利用右键菜单（最推荐）

这是系统自带的功能，但默认可能没有开启。设置一次后，终身受益。

1. 打开 **系统设置** (System Settings)。
2. 导航至 **键盘** -> **键盘快捷键...** -> **服务**。
3. 在右侧列表中找到 **文件和文件夹**。
4. 勾选 **“在文件夹处新建终端标签页”** 或 **“在文件夹处新建终端窗口”**。
5. **如何使用**：在访达（Finder）中，**右键点击**任何文件夹，选择底部的 **服务** -> **在文件夹处新建终端窗口**。

---

### 方法二：拖拽大法（最直观）

如果你已经打开了一个终端窗口，但想快速切换到某个文件夹：

1. 在终端输入 `cd `（注意 `cd` 后面有一个**空格**）。
2. 直接将访达里的文件夹图标**拖入**终端窗口。
3. 终端会自动填充该文件夹的完整路径，按 **回车** 即可。

---

### 方法三：将文件夹拖入程序栏

如果你经常需要针对不同文件夹开启终端，可以将终端图标固定在 **Dock 栏**：

1. 打开访达，选中你想要的文件夹。
2. 按住文件夹并将其拖动到 Dock 栏上的 **终端 (Terminal)** 图标上。
3. 终端会自动启动并直接进入该目录。

---

### 方法四：使用“标题栏”小技巧

在访达窗口中，窗口顶部中间有一个文件夹的小图标（或者是文件夹名字）：

1. 按住 `Command` 键并点击窗口顶部的**文件夹名称**，你会看到路径层级。
2. 其实更简单的做法是：直接**右键点击**窗口顶部的图标或名称，有时可以直接通过自定义工具栏添加“在终端打开”的插件。

---

### 💡 极客推荐：使用路径导航栏

在访达中，点击菜单栏的 **显示** -> **显示路径栏**（或按下 `Option + Command + P`）。
窗口底部会出现当前的路径序列。你可以在底部的任何一级文件夹上**右键点击**，同样能找到“在终端打开”的选项。

需要我帮你设置一个**快捷键**，让你选中文件夹后按一下键盘就能秒开终端吗？


要为“在当前文件夹打开终端”设置一个快捷键，我们需要结合 macOS 的“服务（Services）”和“键盘快捷键”设置。

请按照以下步骤操作：

### 第一步：启用服务

1. 点击屏幕左上角的 **苹果图标 ()**，选择 **系统设置...** (System Settings)。
    
2. 在左侧栏向下滚动并点击 **键盘** (Keyboard)。
    
3. 点击右侧的 **键盘快捷键...** (Keyboard Shortcuts) 按钮。
    
4. 在弹出的窗口左侧选择 **服务** (Services)。
    
5. 点击 **文件和文件夹** 前面的箭头展开选项。
    
6. 找到 **“在文件夹处新建终端窗口”** (New Terminal at Folder)，确保前面的勾选框是**选中状态**。
    

### 第二步：设置快捷键

1. 在同一行（“在文件夹处新建终端窗口”）的右侧，你会看到“无”或者现有的快捷键。双击它。
    
2. 现在直接在键盘上按下你想要的组合键。
    
    - **建议方案**：`Control + Option + T` （因为这个组合通常不会和系统其他快捷键冲突）。
        
3. 点击 **完成** (Done)。
    

---

### 如何使用：

- **方法 A (选中文件夹时)**：在访达中点一下某个文件夹，按下你设置的快捷键，终端就会弹出。
    
- **方法 B (在文件夹内部时)**：如果你已经在文件夹里了，**不要选中任何文件**，直接按下快捷键，它也会在当前目录下打开。
    

---

### ⚠️ 注意事项（常见问题）：

如果按下快捷键没有反应，通常是因为：

1. **焦点不在访达**：确保你当前点击了一下访达窗口。
    
2. **权限问题**：部分系统版本在第一次运行时会弹出警告，询问是否允许，请点击“允许”。
    
3. **快捷键冲突**：如果你设置的快捷键（比如 `Command + T`）已经被浏览器或访达自带的功能占用了，它可能失效。建议使用包含 `Control` 或 `Option` 的组合键。
