这是一个关于逻辑学核心原则的关键问题，即推论的对错（有效性）与经验事实（真假）之间的关系。根据吴先生（逻辑学教授）的详细解释，确定推论的对错不可拿事实做根据，是因为逻辑学关注的是推论的形式，而非其特殊内容或经验语句的真假。

以下是来源中对此问题的全面解释：

### 逻辑推论的核心关注点：形式与规律

1. 逻辑研究推论的效准，而非经验事实的真假：


* **逻辑的本质：** 逻辑是**必然有效的推论规律的科学**。它所涉及的不是经验语句的真假问题，而是**决定哪些规律可以保证推论有效的问题**。
* **推论的对错与语句的真假是独立的：** 逻辑推论的对错与语句的真假是**平行的，各不相干**。语句的真假是关于经验的，而推论的对错是关于**纯粹的逻辑规律**的。
* **推论的普遍妥当性：** 逻辑所要保证的是推论的**普遍的效准**。这意味着逻辑规律必须**对于每一个人情形都真**，不许有例外。

2. 依靠经验判断推论的错误性：


**经验的局限性：** 仅凭经验根本**无推论的把握可言**。在施行推论时，如果依靠经验，不仅不必有助，**有时反而有害**。经验固然可以有助于学习，但会**窒碍逻辑推论**。

**例子一：判断错误且理由不相干：**

* 吴先生曾举例：“一切化学系的学生的都在化学实验室工作，甲组的学生都在化学实验室工作，所以甲组的学生都是化学系学生。” 学生周文璞认为这个推论是**对的**，因为他**听不出什么不合事实的毛病**。
* 吴先生又举例：“一切尼姑都是女性，一切苏州女人都是女性，所以一切苏州女人是尼姑。” 周文璞认为这个推论是**不对的**，理由是“在事实方面，我们知道并不是一切苏州女人都是尼姑”。
* **吴先生的结论：这两个推论都是错误的**，并且它们错误的逻辑地方完全相等。周文璞说第二个推论不对的理由**不相干，不是逻辑的理由**。他正是从**事实上的知识来判断我的推论不对**。
**例子二：形式相同，结论真假偶然：**

* 吴先生举了两个形式完全一样的推论，但前提内容不同：
* 推论一：“一切杨梅是酸味的，香瓜不是杨梅，所以没有香瓜是酸味的。” 周文璞凭经验说它**对**，因为“香瓜都是不酸的”。
* 推论二（把杨梅和香瓜换成橘子）：形式与推论一完全一样，但结论“没有橘子是酸味的”显然是**假的**。
* **吴先生的结论：** 周文璞的判断是**被经验所蔽**。这两个推论形式根本是**错的**。第一个推论结论之真或第二个推论结论之假，都只是**碰巧**的。仅凭经验来判断推论，常常免不了瞎摸乱猜。

3. 逻辑推论只依据推论法则：


* **可靠的办法：我们确定推论之对错，不可拿事实做根据**。唯一的可靠办法是**看它是否合于逻辑推论的法则**。
* **保证推论有效：** 如果推论合于推论法则，那么推论一定是对的；如果不合乎推论法则，那么推论一定是不对的。
* **推论与经验的关系：** 在行严格逻辑推论的时候，推论的对或错**完全以推论法则为依据，不依靠经验或事实**；经验或事实对于纯粹推论**丝毫没有帮助**。

### 真假与对错的独立性

吴先生通过真假与对错的组合，进一步说明了它们是“各自行其是”的独立系统：

|     |     |     |     |     |
| --- | --- | --- | --- | --- |
| 前提真假 | 推论对错 | 结论真假 | 结论对错 | 意义  |
| 真 (T) | 错 (I) | 不定 (?) | 错 (I) | 前提纵真而推论错，则结论不一定真，而是真假不定，但一定错。推论错时，即使结论碰巧为真，在推论场合下也是错的。 |
| 假 (F) | 对 (V) | 不定 (?) | 对 (V) | 如果前提假而推论对，那么结论有时真，有时假，但一定对。推论对时，前提即使是假的，结论一定是对的。 |

由上表可见：

* 语句（前提/结论）的**真假**与推论的**对错**没有必然的关联。
* 推论的**对错**与结论的**对错**有必然的关联：**推论对时结论必对；推论错时结论必错**。

因此，逻辑学必须远离经验，排除经验语句的真假（即事实）的干扰，才能专心研究和保证推论形式的普遍效准。
