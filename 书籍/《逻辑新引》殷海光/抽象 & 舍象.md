简而言之：**“舍象”是“抽象”的前提和方法；“抽象”是“舍象”的目的和结果。**

它们是同一思维活动中不可分割的两面。

* * *

### 详细解释

为了理解这一点，我们先看看“抽象”这个思维过程到底是怎么回事。

“抽象”（Abstraction）的字面意思是“抽取（形象）”。它的目的是从许许多多、各式各样、具体生动的事物中，**提炼出它们共同的、本质的属性**，形成一个“概念”。

那么，你要怎么“提炼”呢？这就需要两个步骤：

#### 1\. 舍象 (Shěxiàng)：“舍弃”和“忽略”

**字面意思：** 舍弃表象或形象。

**思维动作：** 这是一种\*\*“减法”\*\*。

**过程：** 当我们面对一堆具体事物时，会发现它们有无数多的细节。比如，看到张三、李四、王五（都是“人”），他们有不同的身高、体重、肤色、性别、国籍、性格……

**“舍象”就是：** 为了思考“人”这个概念，我们必须\*\*主动地、有意识地忽略（舍弃）\*\*所有这些让张三、李四、王五彼此不同的具体特征（身高、肤色、国籍等）。


#### 2\. 抽象 (Chōuxiàng)：“抽取”和“概括”

**字面意思：** 抽出共同的形象。

**思维动作：** 这是一种\*\*“加法”**或**“提纯”\*\*。

**过程：** 当我们“舍象”了所有那些非本质的差异之后，剩下的共同点是什么？

**“抽象”就是：** 我们把那些“舍弃”不掉的共同属性（比如“会思考”、“能制造工具”、“有社会性”等）**抽取**出来，并把它们**概括**在一起，形成一个清晰的概念，即“人”的本质。


* * *

### 一个简单的例子：形成“水果”的概念

假设一个小孩第一次认识世界，他看到了苹果、香蕉和橙子。

1. **具体事物：**

**苹果：** 红色的、圆的、脆的。

**香蕉：** 黄色的、长的、软的。

**橙子：** 橙色的、圆的、多汁的。

2. **思维过程：**

**舍象（减法）：** 我们必须**忽略**它们的具体颜色（红、黄、橙）、具体形状（圆、长）、具体口感（脆、软、多汁）。

**抽象（加法/提纯）：** 我们**抽取**它们共同的属性（“都是植物的果实”、“可以吃”、“有营养”、“通常是甜的”）。

3. **结果：**

我们得到了“水果”这个**抽象概念**。


### 总结：一枚硬币的两面

没有“舍象”，就无法“抽象”。

如果你不能“舍弃”掉那些五花八门的颜色和形状的干扰，你就永远无法“抽取”出“水果”这个共同概念。

“舍象”是为了“抽象”服务的。

你不是在漫无目的地“忽略”细节，你忽略这些细节的目的，就是为了更清晰地抓住那个“本质”。

在中文哲学，特别是马克思主义哲学的《资本论》的讨论中，“舍象”这个词被特别强调。例如，为了得到“价值”这个抽象概念，马克斯必须“舍象”掉所有商品的具体“使用价值”（衣服能穿、面包能吃）。

### 区别对比表

|     |     |     |
| --- | --- | --- |
| **特征** | **舍象 (Shěxiàng)** | **抽象 (Chōuxiàng)** |
| **字面含义** | 舍弃表象 (Discard surface image) | 抽取共象 (Extract common image) |
| **思维动作** | 忽略、舍弃、做“减法” | 概括、提取、做“提纯” |
| **关注点** | 忽略“差异点”、“非本质属性” | 关注“共同点”、“本质属性” |
| **在过程中的角色** | **手段 / 前提** | **目的 / 结果** |
| **比喻** | 滤掉渣滓（非本质） | 得到纯液（本质） |
