### 最常见的 toggle 命令（基本都内置）

1. **Toggle live preview / source mode**  
   中文命令名：**切换实时预览/源码模式**（或类似“Toggle source mode” / “Switch between Live Preview and Source mode”）  
   - 默认快捷键：**Ctrl/Cmd + E**  
   - 作用：在当前笔记的**编辑状态**下，快速在“实时预览”（所见即所得）和“纯源码”（Markdown 文本）之间切换。  
   - 这就是你之前提到的“按 Ctrl/Cmd + E 切换到源码模式”的那个命令。  
   - 命令面板搜索：输入“toggle source” 或 “切换源码” 就能找到。

2. **Toggle reading view / editing view**（整体阅读 ↔ 编辑）  
   中文：**切换阅读/编辑视图** 或 **Toggle preview**  
   - 快捷键：没有全局默认，但很多人自定义为 **Ctrl/Cmd + E**（早期版本默认是这样，后来改了）  
   - 作用：让整个笔记在“阅读模式”（纯预览）和“编辑模式”之间切换。  
   - 如果你开了 Live Preview，这个命令和上面第1个会有些重叠行为。

3. **其他非常常见的 toggle 类命令**（直接在命令面板搜 “toggle” 就能看到一大堆）：
   - **Toggle left sidebar** → 左侧边栏显示/隐藏（很多人自定义为 Ctrl + \ 或 Ctrl + B）
   - **Toggle right sidebar** → 右侧边栏显示/隐藏
   - **Toggle pin** → 当前标签页固定/取消固定
   - **Toggle fold on current line** → 当前行（标题或列表）折叠/展开
   - **Tasks: Toggle task done**（如果装了 Tasks 插件）→ 任务复选框打勾/取消
   - **Toggle spellcheck** → 拼写检查开/关
   - **Toggle strikethrough / highlight / bold** 等格式切换（选中文字后执行）
