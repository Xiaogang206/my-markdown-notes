根据来源材料，智慧和有智慧的人具有以下关键特征和作用：

### 一、 有智慧的人是什么样子的？

来源明确区分了“聪明人”和“有智慧的人”，并指出智慧的人拥有独特的品格和认知：

1. **谦卑（謙卑）**：一个有智慧的人**一定是一个谦卑的人** 。
2. **承认无知**：有智慧的人**承认自己的无知**，因为这是**开启智慧的大门** 。
3. **认知局限性**：他们越是站立在人类历史的巅峰，越是望向远方，越会发现自己**是如此的渺小** 。他们认识到自己只是看到了**知识的惊鸿一瞥**，还有很多东西是自己不知道的 。
4. **区别于聪明**：世界上有许多人很聪明，**但是聪明人不一定有智慧** 。

总而言之，这个世界缺的是有智慧的人 。

### 二、 智慧有什么用？（智慧的益处）

智慧不仅是心智的品质，它对于个人修养和社会交往具有重要的作用：

1. **作为阅读的目标**：读书的目的是为了**培养你的智慧** 。
2. **遏制优越感**：智慧（通过认识到自身的局限）可以**遏制你内心的知识优越感** 。
3. **避免狂妄自大**：它可以帮助一个人**避免狂妄自大** 。
4. **发自内心地尊重他人**：只有这样（具备谦卑和对知识局限性的认识），才能让你**真正地发自内心地去尊重他人** 。

简而言之，智慧的作用在于培养谦卑的品格，并以此为基础，瓦解知识优越感，从而实现对他人发自内心的尊重 。
