根据来源中吴先生（逻辑学教授）的观点，关于“旧的东西”（尤其是指代传统、历史或古典概念），逻辑学和哲学在性质、发展速度以及对待旧有概念的态度上存在显著差异：

### 1\. 逻辑学（Logic）对“旧的东西”的态度

逻辑学对待旧有概念和传统逻辑的态度是批判性、修正性和技术性的。

* **技术进步迅速**：逻辑学中的**技术性**进步比基本观念的进步更为**急速**。例如，现在的小学生会算的算术，在古代需要大数学家才能解决。
* **传统逻辑的局限与错误**：
* **传统逻辑**（源于亚里士多德）在 19 世纪中叶以前，一直局限于一个**狭小的范围**，主要以研究**主宾词式的语句**为中心。
* 传统逻辑没有将**关系的研究**包含进去，这限制了其内容的丰富程度。
* 许多**传统教科书**中的说法，照现代逻辑的眼光看来是**说不通的**。这些传统说法常常混杂了**语法、心理学、知识论、伦理学甚至形上学的成分**。
* **现代逻辑的更新**：现代逻辑要求**与时俱进**，放弃那些错误的旧有说法。逻辑通过**符号化**（如乔治·布尔的贡献）进入了新阶段，使得逻辑的性质大白。
* **对“三大思想律”的重估**：传统的“三大思想律”（同一律、矛盾律、排中律）被认为**须重新估量**。在《数学原理》（被视为亚里士多德《工具论》之后的里程碑）等现代逻辑系统中，这些定律根本**不重要**，只是被推论出来的语句，因为它们在记号结构上**不够丰富**，缺乏衍之力。逻辑家指出，将这些律视为\*\*“关于事物”**的规律是毫无意义的，它们实际上是**语法或语意之最低限度的必须条件\*\*。

因此，逻辑学是不断自我批评和技术改进的科学，许多“旧的东西”在现代逻辑看来，是需要修正甚至抛弃的，以追求形式的纯粹性和普遍的效准。

### 2\. 哲学（Philosophy）对“旧的东西”的态度

哲学对待“旧的东西”的态度则不同，新的不一定更好。

* **原创能力持久**：吴先生认为，哲学的**原创观念**固然不免受到修正，但修正的速度**不怎么急速**。
* **古代哲学家的地位**：过去的大哲学家有许多**原创能力**（originality），吴先生不相信现在有太多人能够**超过柏拉图、亚里士多德、休谟、康德**的原创能力。
* **研究范围的差异**：逻辑学在历史上曾与哲学混在一起。但现代逻辑主张将自身与**形上学、知识论**等哲学题材划清界限。研究哲学没有像研究科学那样受到实验、客观事物或符号语言的**限制**，因此弄哲学常常容易弄得**漫无边际、言人人殊**。

简而言之，对于“旧的东西”：

|     |     |     |
| --- | --- | --- |
| 方面  | 逻辑学（现代逻辑） | 哲学  |
| **价值判断** | 许多旧概念和传统说法被视为**错误或粗糙**，需要用现代技术**修正或淘汰**，以追求**严谨**和**有效性**。 | 新的不一定更好，古代大哲学家（如柏拉图、亚里士多德）的**原创能力**和基本观念仍然具有重要地位。 |
| **核心关注点** | 形式结构、推论规律的**普遍效准性**和**技术精度**。 | 对事物和世界的**基本观念**和**原创性**的探索。 |
| **发展速度** | 技术性进步**急速**。 | 基本观念的进步**不急速**。 |
