## Grok
很多人从<mark style="background:#d3f8b6">印象笔记（Evernote）切换到 Obsidian </mark>后都会经历一段非常不适应的时期，这几乎是普遍现象。原因主要集中在下面这几点：

- 印象笔记是“富文本 + 所见即所得 + 中心化数据库”模式  
- Obsidian 是“纯文本 + Markdown + 文件夹 + 双向链接”模式

这两种范式差异很大，就像从用Word突然全部改用VS Code写东西一样。

下面给你一些最务实、短期见效的过渡建议，按优先级排序：

### 1. 先把「视觉 & 操作习惯」对齐（最快降低不适感）
做完这几步，打开Obsidian时至少不会觉得“完全陌生”

| 要做的事                 | 怎么做                                             | 能解决的不适点        |
| -------------------- | ----- | -------------- |
| 换成浅色主题               | 设置 → 外观 → Theme → 选一个明亮主题（如Things、Minimal调整为浅色） | 印象笔记默认白底太刺眼了   |
| 开启所见即所得编辑器           | 社区插件 → 安装 Editing Toolbar + Style Settings      | 接近印象笔记的工具栏体验   |
| 打开实时预览（Live Preview） | 设置 → 编辑器 → 默认使用实时预览                             | 写的时候不用一直切源码/预览 |
| 把侧边栏调到印象笔记风格         | 左侧放文件列表，右侧放大纲/反向链接/标签                           | 熟悉的布局感         |
| 字体调大一点               | 设置 → 外观 → 字体大小 调到18–20px                        | 阅读更舒适          |

很多人调完这几项后，心理落差会立刻减小一半。

### 2. 接受“双链笔记 ≠ 印象笔记的升级版”，而是另一种范式
最容易卡住的新手思维是：  
“我要把印象笔记那一套完全搬过来”

但其实应该反过来想：

- 印象笔记擅长：收集、堆积、搜索、随手记
- Obsidian 擅长：连接、提炼、长期思考、知识网络

所以初期建议：
- 继续把印象笔记当作“收件箱 + 收集器”
- 只把真正想深度消化的内容，才搬到Obsidian里建立链接
- 不要强求“全部迁移后立刻删除印象笔记”，很多人用了1–2年才逐步关掉印象笔记

### 3. 最实用、马上能缓解不适的几个插件（按推荐顺序）
先装这几个就够用了，别一上来装二三十个

1. **Editing Toolbar** → 出现熟悉的加粗、列表、表格工具栏
2. **Style Settings** → 配合主题微调颜色、间距
3. **Advanced Tables** → 表格体验接近印象笔记
4. **Reminder** 或 **Tasks** → 代替印象笔记的待办
5. **Obsidian Importer**（如果还没导完）→ 批量处理印象笔记残留
6. **Paste Image Rename** → 粘贴图片自动重命名，更整洁
7. **Homepage** → 打开软件直接进入每日笔记，像印象笔记主页一样

### 4. 典型不适点 & 对应解法速查表

| 不适表现                       | 根本原因                       | 推荐做法                                   |
|-----------|-----------|--|
| 找不到以前的笔记               | 路径/搜索习惯不同              | 装 Dataview + 用标签/文件夹双轨制过渡     |
| 写字没以前爽                   | Markdown太简陋                 | 开启实时预览 + 装Editing Toolbar           |
| 网页剪藏体验差                 | 官方剪藏器很弱                 | 用 MarkDownload / Obsidian Web Clipper     |
| 附件/图片管理乱                 | 默认放笔记同目录               | 设置统一附件文件夹 + 装 Attachment Management |
| 总想回去用印象笔记             | 旧习惯太强                     | 允许“双开”3–6个月，慢慢把高价值内容迁移   |

### 5. 给你一个最温和的过渡路径（很多人走通了）
第1–4周：只用Obsidian写新东西 + 每日笔记，旧笔记还在印象笔记  
第5–12周：把最常翻的20–30%笔记迁移过来，建立链接  
3–6个月后：真正的高质量内容基本都在Obsidian，印象笔记只剩历史档案  
6–12个月后：大部分人会自然冷掉印象笔记

最后想说一句很实在的话：

**不习惯是正常的，坚持3个月以上才算真正“试用”过Obsidian。**  
很多人第2–4周最想放弃，但熬过去后反而回不去印象笔记了。
