根据来源的讨论，\*\*极权主义（Totalitarianism）与威权主义（Authoritarianism）\*\*是中国古代政治制度和现代政治形态比较中的两个重要概念，它们在权力的集中程度、统治基础、以及与专制主义的关系等方面存在显著区别。

### 一、 极权主义 (Totalitarianism) 的性质与特征

极权主义（Totalitarianism）的英文词根是 “total”，意为\*\*“包圆儿”、“全部”\*\*。

* **权力集中程度：** 极权主义意味着**一切权力归于某个人，或者某个小群体**。
* **与专制主义的关系：** 极权主义可以被看作是**专制主义（Despotism）的顶峰**。

来源的观点表明，极权主义是一种极致的、全方位控制的统治形式。

### 二、 威权主义 (Authoritarianism) 的性质与特征

威权主义（Authoritarianism）的英文单词是 “authority”，意为权威。

* **统治基础：** 威权主义是由**一个人，或者一小撮、一小群人**来实行统治的。这些人**应具有崇高的威望**，例如**打江山的那批人**，他们的威望本身是不可替代的。
* **统治性质：** 它实行的是**偏向于专制的统治**。
* **制度定位：** 威权主义**介乎于民主和专制之间的一种东西**。来源认为，威权主义**不一定算集权**，它与专制也**不完全一样**。
* **实例：新加坡的李光耀**所建立的体制，就被认为是**有点威权主义**。

### 三、 极权主义与威权主义在制度选择中的讨论

在政治制度与社会形态的背景下，来源提到了李光耀为威权统治辩护的观点，并对其进行了反思：

* **李光耀的观点：** 李光耀（包括他的儿子李显龙）认为民主制度不一定适合所有国家，必须**结合当地的经济和国民素质**，以此为新加坡建立的威权体制找借口。
* **来源的反思：** 来源承认，在国家**初创**或者处在**危机存亡之秋**时，民主的弊端会很明显，特殊情况可以**特殊处置**。但问题是国家**不能永远是初创阶段或危机之中**。在绝大多数时候，国家**还是要还政于民**，让老百姓真正当家做主。

### 四、 区分中央集权 (Centralization)

来源也区分了另一个容易混淆的概念——中央集权（Centralization），这与极权主义有所不同：

|     |     |     |
| --- | --- | --- |
| 概念  | 英文名称 | 含义与特点 |
| **极权主义** | Totalitarianism | **极致的**权力集中，一切权力归于个人或小群体，可视为**专制主义的顶峰**。 |
| **中央集权** | Centralization | **权力汇总**，关注**地方服从中央**，地方官由中央任免。中央集权**不一定反民主**，也不一定就是坏的。 |

来源指出，中国古代的基本政治特色是专制主义中央集权（Despotism Centralization），即中央集权与专制主义相结合，毫无疑问是反民主的。

### 总结

在政治制度演变的脉络中，极权主义代表了权力控制的最极端、最全面的形态（Totalitarianism，专制之顶峰）。而威权主义则是一种基于个人或小群体权威的、偏向专制但尚未达到极权程度的统治形式。尽管威权主义在某些国家（如新加坡）被用于为政治过渡或发展寻找理由，但来源强调，其合法性应是暂时的，最终国家仍须回归到还政于民的民主进程上来。
