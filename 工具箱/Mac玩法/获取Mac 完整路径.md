在 Mac 上获取文件夹的**完整路径**（绝对路径，比如 `/Users/你的用户名/Documents/Books`）非常简单，以下是几种最常用、最快捷的方法，按实用度排序（2026 年 macOS 最新系统如 Sequoia / Tahoe 都适用）：

### 1. 最推荐：快捷键复制路径（一键搞定）
- 在 Finder（访达）中选中你要获取路径的文件夹（或文件）。
- 按下 **Option（⌥） + Command（⌘） + C**。
- 这会直接把文件夹的**完整 POSIX 路径**复制到剪贴板（类似 `/Users/你的用户名/Downloads/MyEPUBs`）。
- 然后去任何地方（如 Obsidian 的 Epub Importer 输入框、终端、文本编辑器）按 **Command + V** 粘贴即可。

> 这个快捷键是 macOS 原生支持的，几乎所有版本都有，超级方便。

### 2. 右键菜单方式（不用记快捷键）
- 在 Finder 中选中文件夹。
- 按住 **Option（⌥）** 键的同时，右键（或 Control + 点击）该文件夹。
- 菜单中原本的“拷贝”会变成 **“拷贝‘文件夹名’为路径名称”**（或类似“将 XX 拷贝为路径名”）。
- 点击它 → 路径就复制到剪贴板了。
- 然后 Command + V 粘贴。

### 3. 通过终端（Terminal）拖拽（最直观，适合不确定路径时）
- 打开 **终端**（Terminal.app，可以通过 Spotlight 搜 “终端” 打开）。
- 在终端窗口里输入 `cd `（cd 后加一个空格，不要回车）。
- 把 Finder 中的文件夹直接**拖拽**到终端窗口里。
- 松手后，终端会自动显示完整路径（如 `/Users/你的用户名/Books`）。
- 你可以选中它 → Command + C 复制。
- 或者直接回车执行 cd 命令，之后用 `pwd` 命令再确认当前路径。

### 4. 显示路径栏查看（适合经常需要看路径）
- 打开 Finder。
- 菜单栏 → **显示（View）** → **显示路径栏**（快捷键：**Option + Command + P**）。
- Finder 窗口底部会出现路径栏，从根目录一路显示到当前文件夹。
- 右键路径栏中最右边的文件夹图标/名称 → 选 **“拷贝为路径名称”** → 就复制完整路径了。

### 5. 显示简介里看（备用，不推荐频繁用）
- 选中文件夹 → 右键 → **显示简介**（或 Command + I）。
- 在弹窗的“通用”部分，看“位置：”后面的路径（但不是最完整的 POSIX 路径，有时是简化版）。
- 可以手动选中复制。

### 小贴士（针对 Obsidian Epub Importer）
- 导入 EPUB 时，需要的是**文件夹的完整路径**（如果你的 EPUB 存放在某个文件夹里），直接用上面第 1 或第 2 方法复制路径，粘贴到插件的输入框。
- 示例路径格式（Mac 常见）：
  - `/Users/你的用户名/Downloads/我的书.epub`（单个文件）
  - `/Users/你的用户名/Documents/EPUB Library/`（文件夹路径，插件如果支持批量）

这些方法都不需要安装任何额外软件，纯系统原生。如果你用的是中文 macOS，菜单可能会显示为“拷贝为路径名”或类似，操作完全一样。
