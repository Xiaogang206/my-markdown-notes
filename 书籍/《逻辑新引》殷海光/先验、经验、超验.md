您提出了关于先验、经验和超验这三个概念的哲学分类及其相互关系的深入问题。虽然您的来源并未明确断言这三者是否“穷尽”了所有知识类型，但通过对科学认知活动及其边界的分析，我们可以清晰地界定这三个概念的本质、差异，以及它们在知识论中的位置。

根据来源的论述，这三个概念代表了认知活动中对于知识来源和性质的不同划分：

* * *

### 一、 三个概念的界定与异同

|     |     |     |     |
| --- | --- | --- | --- |
| 概念  | 知识来源/性质 | 确定性/可靠性 | 来源依据 |
| **先验** (A priori) | **不依靠感官经验而成立的知识**。 | **约定的确定性**。经解析，无一不是**约定俗成之结果**。属于**固体似的知识**（如数学、逻辑、理论物理学），具有**必然性**。 | 逻辑、数学。 |
| **经验** (Empirical) | 依靠**感官经验**，通过**观察、试验和推广**来建立。 | 只有**盖然性**（probable），无法达到必然。知识的可靠性具有**程度差别**。 | 物理学、地质学、生物学、经济科学等**经验科学**。 |
| **超验** (Transcendent) | **超乎一切可能检证**的信仰或言论。与可经验项无关。 | **不具**事实科学语句所有的那种类型的意义。属于非认知的、情感或形上学的范畴。 | 超越的神学、超越的形上学。 |

#### 关键差异点：

1. **对经验的依赖性：**

* **先验**知识**不依赖**经验（不靠感觉经验而成立）。
* **经验**知识**依赖**经验（大部分依赖观察、试验和推广）。
* **超验**概念**超越**经验（超乎一切可能检证）。
2. **确定性的本质：**

* **先验**知识（如纯逻辑）追求**必然的**确定性，但这种确定性是**约定的**。
* **经验**知识寻求**可靠的**和**精确的**知识，但只能达到**盖然的**确定性。
* **超验**概念不追求或不具备认知的确定性，它们缺乏科学知识所要求的**互为主观的可检证性**。
3. **科学的定位：**

* 科学（认知活动）只研究**可经验项**，它寻求具有**纯形式的意义**（先验/逻辑）和**经验的意义**（经验科学）的知识。
* 科学必须将**超乎一切可能检证**的**超越的神学和超越的形上学**排除在外，因为这些言论不具有事实科学语句所有的那种类型的意义。

* * *

### 二、 关于概念的穷尽性

来源将知识的类型主要划分在以下几个方面：

1. **形式的意义（接近先验）：** 逻辑与数学。
2. **经验的意义（经验）：** 化学、地质学等经验科学。
3. **科学以外/非认知（接近超验）：** 神学、形上学、神秘主义、直觉和辩证法等。

虽然来源没有明确使用“穷尽”一词来总结这三类，但它通过设定科学的边界，实际上涵盖了知识获取的两种主要途径（基于理性的形式结构和基于感官的经验事实），并将所有无法通过这两种途径进行验证的领域（即超验的领域）排除在了认知知识之外。

因此，从严格的逻辑和科学认知的角度来看：

* **先验**和**经验**知识（即具有**认知意义**的知识）的结合，构成了科学所能接受的全部知识体系。
* **超验**概念（例如超越的神学和形上学）被视为在知识论上**非逻辑的**或**非经验的**，它们不具备常识语句或事实科学语句所拥有的那种类型的**意义**，因此位于认知领域的界限之外。

这些分类区分了必然的、约定的真理（先验）、盖然的、基于事实的真理（经验），以及超乎验证的、情感或信仰的领域（超验）。

我们可以将这三者想象成一个建筑师的设计图纸：**先验**是纯粹的几何规则和数学公式，保证了结构在概念上的**必然有效**；**经验**是根据土壤、气候、材料等实际**环境数据**来调整设计，保证了建筑在现实中**盖然可靠**；而**超验**，则可能是关于这座建筑应承载的\*\*“永恒精神”**或**“绝对实在”\*\*的信仰，这些概念虽然宏大，但却无法被纳入实际的工程计算和结构验证中。
