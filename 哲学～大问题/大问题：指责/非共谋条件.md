# 谨言慎行派

在“谨言慎行派”（Prudent School）的道德哲学框架下， **“非共谋条件”（Non-complicity condition）** 被视为获取“指责资格”（Standing to Blame）的四个核心支柱之一,。

根据你提供的来源，以下是“谨言慎行派”对这一条件的具体看法与论证逻辑：

### 1. 核心定义：指责者不能是错误的一环

“谨言慎行派”认为，如果你对被指责者的错误负有责任，或者在其中起到了**推波助澜**的作用，你就失去了指责对方的资格。在这种情况下，指责不再是正义的伸张，而是一种 **“甩锅”行为**——通过聚焦他人的错误来掩盖自身尚未处理的过失。

### 2. 核心论据：道德平等与规范性呼吁

该派别通过引用哲学家托马斯·斯坎伦（Thomas Scanlon）的观点来论证这一条件的必要性：

- **相互问责的承诺**：道德指责本质上是一种“规范性的道德呼吁”。当你指责他人时，你实际上是在宣示一项包括你在内的所有人都应当遵守的道德规则。
- **豁免权的矛盾**：如果指责者本身就是共谋者，却拒不承认或修复自己在错误中的角色，他实际上是在要求他人承担道德压力，却**豁免了自己**。
- **否认平等地位**：这种“选择性指责”违反了人与人之间的道德平等原则，因为它默认指责者可以站在道德高地上而不必受同样的规则约束。

### 3. 典型案例分析：以《绝命毒师》为例

来源通过美剧《绝命毒师》中老白（Walt）与妻子（Skyler）的关系深刻说明了这一点：

- **事实背景**：老白从事制毒等严重犯罪，并拒绝签署离婚协议，将家庭置于危险之中，导致妻子为了摆脱他而选择出轨,。
- **派别观点**：谨言慎行派认为，老白**完全没有资格**指责妻子出轨。因为妻子的行为很大程度上是被老白的行为“逼出来的”，老白正是导致这一过错局面的重要一环。

### 4. 道德实践的要求：先对自己负责

“谨言慎行派”强调，共谋并非普通的道德瑕疵，它触及了指责的正当性根基：

- **处理自身责任是前提**：在一个人处理好自身在错误中扮演的角色之前，他对他人的指责是不正当的。
- **防止道德实践的异化**：如果不坚持非共谋条件，允许共谋者在不承担自身责任的前提下继续责备他人，会削弱道德实践中 **“先对自己负责”** 的基本要求。

### 总结

在“谨言慎行派”看来，非共谋条件是为了防止指责被异化为一种**防御机制或转移责任的手段**。从这个视角看，一个参与了错误行为的人，在指责另一方时，实际上是在利用道德语言来掩盖自己的道德亏欠，这种行为本身就是对道德严肃性的损害,。

# 放任指责派

在“**放任指责派**”（反驳限制条件的一方）的视角下，他们坚决反对“指责者必须清白（非共谋）”的预设。针对你提到的“**污点证人：共谋者揭露错误更具深度**”这一观点，来源中提供了以下几个核心层面的论证：

### 1. “污点证人”的独特视角：揭露黑幕的深度

放任指责派提出了一个极具说服力的类比——**法庭上的污点证人**。

- **内部人的洞察力**：一个参与过错误行为的人（共谋者），往往比道德清白的旁观者更清楚该行为的运作机制、危害以及后果。
- **揭露黑幕**：来源以“电信诈骗集团成员”为例。虽然此人参与了诈骗，理应受罚，但他跳出来指责电诈行为时，由于他曾身处其中，他能给公众揭露更多**外部人无法触及的黑幕**，从而让更多人避免上当。这种指责的“深度”是普通旁观者无法提供的。

### 2. 道德改进的实用主义：教育与激励价值

放任指责派认为，指责的价值不应取决于指责者是否完美，而应看其是否产生了积极的社会后果：

- **跨越“完美”陷阱**：如果非要等到自己完美无缺才开口，那么很多系统性的恶行将永远无人揭露。
- **公共教育意义**：即便指责者有缺陷，他的批评依然能让被指责者和旁观者从中学习，理解错误所在。只要这种指责能引导大众**反思行为、纠正偏差**，它就具有不可替代的道德价值。

### 3. 一种“更成熟的道德态度”

放任指责派认为，一个共谋者的指责如果伴随着自我反省，反而体现了更高阶的道德担当：

- **承认不足与关注他人并行**：一个人可以一边承认并修正自己的错误，一边指出他人的不当。这两者并不冲突，反而展现了其**推动整体道德改善**的愿望，而非单纯的“甩锅”。
- **论事不论人**：该派别强调应遵循“**论迹不论心**”的原则，将注意力集中在“被指责的事实是否错误”上，而不是通过攻击指责者的身份（即他也是共谋者）来转移焦点、掩盖恶行。

### 4. 反驳“非共谋条件”的底层逻辑

放任指责派指出，谨言慎行派所坚持的“先对自己负责”虽然在直觉上有吸引力，但在现实中会产生负面影响：

- **防止人身攻击**：如果每一个指责都能因为指责者曾是“一环”而被轻易驳回，那么道德讨论将不断滑向**人身攻击和身份审查**，而真正的错误行为反而会被置之不理。
- **不因人废言**：即使指责者本身是可恶的共谋者，他所指出的事实如果是真实的，社会就不应拒绝认真对待这些信息。

**总结：** 在放任指责派看来，共谋者的指责不仅具有正当性，更因为其 **“内部人”的身份**而具备了普通指责所欠缺的深度与广度。他们认为，与其纠结于指责者的资格，不如利用这些“污点证人”提供的信息来**捍卫公共规范、教育共同体成员**并最终实现道德的整体进步。
