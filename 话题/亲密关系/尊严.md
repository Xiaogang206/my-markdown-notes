## 👑 什么是尊严？

“尊严”（Dignity）是一个深刻的哲学、道德和法律概念，它指的是**一个人或事物内在固有的价值，这种价值不依赖于任何外部因素或成就，因此值得被尊重和珍视**。

简单来说，**尊严是****一个人作为人类（或个体）存在的内在价值和应得的尊重****。**

* * *

### 1\. 尊严的核心要素

尊严的定义可以分解为以下几个关键要素：

#### A. 内在价值（Inherent Worth）

**与生俱来：** 尊严不是通过努力、财富、地位或权力获得的，而是**作为****一个人就天然拥有的**。

**不可剥夺：** 任何外部力量，包括政府、社会、疾病或贫穷，都不能消除或剥夺一个人的尊严。


#### B. 尊重主体性（Respect for Autonomy）

**自我主宰：** 尊严要求个体能够**自主决定**自己的生活、信仰和行为，只要不侵犯他人的权利。

**被视为目的，而非工具：** 康德哲学认为，尊重人的尊严，就是将人永远视为**目的**本身，而不是仅仅当作达成他人目的的**工具**。


#### C. 不可量化性（Non-quantifiable）

**平等性：** 所有人的尊严都是**平等的**，无论其社会地位、种族、性别、能力或贡献如何。

**独立于成就：** 一个人的尊严不会因为他犯了错误、失去了财富或生病而减少。


### 2\. 尊严在不同领域的体现

|     |     |
| --- | --- |
| **领域** | **尊严的体现** |
| **道德/哲学** | 要求人自重自爱，有**羞耻心**和**自豪感**，行为不卑不亢。 |
| **法律/人权** | 它是**人权的基础**。法律要求保护公民免受酷刑、奴役、羞辱或任意逮捕，这些都是对人的尊严的侵犯。 |
| **社会** | 要求社会体系在分配资源和制定政策时，确保每个人都能有**体面**地生存的条件，如享有基本医疗和住房。 |
| **临终关怀** | **临终尊严**指的是在生命末期，病人能够有权决定自己的治疗方式，减少痛苦，并保有其个性和身份直到生命的最后一刻。 |

### 3\. “有尊严”与“没尊严”

**有尊严：** 意味着一个人**自重**，在困境中保持自爱和正直；同时也意味着一个人**被尊重**，没有受到侮辱、贬低或工具化。

**没尊严（或被剥夺尊严）：** 意味着受到**羞辱**、**凌辱**、**奴役**，或者被社会视为**低劣、可替代的工具**。


**总结：**

尊严是人类最基本的精神需求。它不仅要求我们**尊重他人**的内在价值，也要求我们**自重**，不做贬低自身价值的事情。
