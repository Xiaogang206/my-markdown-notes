根据来源中的信息，深阅读（Deep Reading）是“大阅读”概念的三个重要特质之一，而五个步骤是实现深阅读效果的具体方法论，旨在培养读者的思维深度和输出能力。

### 深阅读方法论的五个步骤

深阅读并非要求对所有书籍都精读，而是针对那些特别值得深读的书籍，采取的一系列有步骤的训练方式。这套方法应该从小孩子开始就要训练。

这五个步骤是一个小的循环，旨在用输出来促进输入：

|     |     |
| --- | --- |
| 步骤  | 具体内容与要求 |
| **第一步：**<br>**阅读并复述** | 要求读者在**合理的阅读速度**下，阅读**一个完整的章节**或几个相互联系紧密的章节。随后，要求孩子进行**复述**（paraphrase），即**告诉他人你读到了什么**。这一步的目的是检验读者是否**真的读进去了**，是否**读权了**（读全了）。 |
| **第二步：**<br>**提出问题** | 提问题被认为是**未来学习中最重要的能力之一**。要求孩子根据笔记内容，提出疑问，尤其是针对与已有知识体系冲突、书内观点矛盾或需要补充的地方。来源特别推崇能够体现思维深度的**复杂疑问句**，即用 **Why**（探究底层原因）、**How**（寻求解决方案/方法）和 **What if**（进行案例假设分析/假如换一个条件会怎么样）开头的提问。成人（老师/家长）应该**鼓励和夸奖**问出好问题的行为，即使问题没有答案。 |
| **第三步：**<br>**讨论**<br>**Discussion** | 根据孩子提出的问题和阅读的内容，进行**讨论**。讨论非常重要，因为不同的人因其人生阅历和背景不同，对**同一文本**会产生不同的理解。这种交流讨论极有益处，甚至**十几岁的孩子**可能会提出**匪夷所思的新视角**。 |
| **第四步：辩论Debate**<br>**和展示Presentation** | 讨论的升级版是**辩论**（debate），当各持己见、到了极致时，可以进行辩论，以实现\*\*“理越辩越明”**。辩论结束后，**展示（presentation）是深阅读非常重要的一个步骤\*\*。展示同样是为了**用输出来促进输入**。来源建议可以增加**仪式感**，例如让孩子在所有人面前进行展示，以避免其对阅读心存懈怠。 |
| **第五步：**<br>**重读文本** | （在第四步之后）可以**重读文本**，或说为**展示**做准备。 |

深阅读的特点是不求速度，它与通常的快速泛读（如成年人每分钟约 500 字的阅读速度）不同。

### 深阅读在大阅读方法论中的定位

“大阅读”被视为终身学习的根本解决之道，也是和古人神交的最便捷方式。它包含三个特质：数量要大、经典型阅读（质量要高）、以及深阅读。

深阅读作为大阅读的第三个要素，是认知提升的关键：

* 如果父母处于相对偏远、条件不够好的地区，通过做到大阅读的**数量、质量和深阅读**这三个要素，孩子在认知上\*\*“有可能跟大城市的孩子同步起飞，甚至要碾压他们”\*\*。
* 深阅读是**思维能力的训练**的具体实践，特别是批判式思维、全局观和逻辑。在深阅读的第二步中，通过提出复杂的 Why, How, What if 问题，可以培养和检验孩子的**思维深度**。

来源建议，最好的学区房就是家里的书房，通过有规律地进行大阅读，并与行路相结合（例如观看高质量纪录片），可以开阔孩子的眼界，缓解教育焦虑。
