在传统逻辑学中，您所指的三段式（定言三段式）包含三个核心名词，它们是三段式得以成立和进行推论的基础。

这三个名词分别是：

1. **小词** (Minor term)
2. **大词** (Major term)
3. **共词** (Common term)，也称为**中词** (Middle term)。

在符号表示中，这三个名词通常依次以 G、H 和 M 来代表。

### 名词的结构与作用

在三段式（由大前提、小前提和一个结论构成）的结构中，这三个名词有着特定的位置和作用：

|     |     |     |     |
| --- | --- | --- | --- |
| 名词  | 符号  | 在三段式中的作用 | 在结论中的位置 |
| **共词** | M   | 必须在**大前提**中出现一次，在**小前提**中出现一次。它的作用是**介系**（联系）大词与小词，使两者发生关联。 | **永远不在结论中出现**。 |
| **大词** | H   | 在**大前提**和**结论**中各出现一次。 | **一定是宾位词端**。 |
| **小词** | G   | 在**小前提**和**结论**中各出现一次。 | **一定是主位词端**。 |

### 关于名词数量的澄清

虽然一个标准的三段式包含三个语句（大前提、小前提、结论），且每个语句通常有两个词端，总计有六个词端出现（记号事件），但逻辑学在分析三段式时所关注的是概念（记号设计）：

* 如果一个名词的记号设计（即概念）只有**一个**所指或意谓，那么它即使出现多次，也只算作**一个名词**。
* 因此，尽管大词、小词和共词各出现两次，它们仍然只被计算为**三个名词**。如果它们在两次出现中所指的意义完全相同，则只算作一个名词。如果三段式中没有共词 M，那么小词与大词就无法发生关联，推论也就不能成立。
