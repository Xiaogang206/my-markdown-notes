在 Obsidian 中设置标题颜色，通常有三种主流方法。由于你使用的是 Obsidian，我建议优先考虑前两种，因为它们更符合“笔记系统”的可维护性。

---

### 方法一：使用 Style Settings 插件（最推荐，无代码）

这是最简单、最直观的方法，适合不想折腾代码的用户。

1. 在 **插件市场 (Community Plugins)** 搜索并安装 \`Style Settings\`。
    
2. 确保你使用的是支持该插件的主题（如默认主题、Minimal、AnuPpuccin 等）。
    
3. 进入 `设置 -> Style Settings`。
    
4. 找到 **Heading** 或 **Typography** 选项。
    
5. 展开后，你可以分别为 H1 到 H6 设置不同的颜色、字体大小和粗细。
    

---

### 方法二：使用 CSS Snippets（最专业，一劳永逸）

如果你想精确控制颜色，且不依赖特定主题，可以使用 CSS 代码片段。

1. 打开 `设置 -> 外观 (Appearance)`。
    
2. 下拉找到 **CSS 代码片段 (CSS snippets)**，点击右侧的文件夹图标。
    
3. 在打开的文件夹中创建一个新文件，命名为 `heading-color.css`。
    
4. 将以下代码粘贴进去并保存：
    

CSS

```
/* 设置各级标题颜色 */
.markdown-rendered h1, .cm-header-1 { color: #ff4500 !important; } /* 橙红色 */
.markdown-rendered h2, .cm-header-2 { color: #2e8b57 !important; } /* 海绿色 */
.markdown-rendered h3, .cm-header-3 { color: #1e90ff !important; } /* 闪闪蓝色 */
.markdown-rendered h4, .cm-header-4 { color: #daa520 !important; } /* 金菊色 */
.markdown-rendered h5, .cm-header-5 { color: #8a2be2 !important; } /* 紫罗兰 */
.markdown-rendered h6, .cm-header-6 { color: #5f9ea0 !important; } /* 军兰色 */
```

5. 回到 Obsidian 的 CSS 代码片段设置界面，点击**刷新**，然后**启用**该文件。
    

> **注意：** `.markdown-rendered` 对应阅读模式，`.cm-header-x` 对应实时预览模式。`!important` 是为了防止被主题自带的样式覆盖。

---

### 方法三：使用 HTML 标签（针对单篇笔记或单个标题）

如果你只想让某一个标题变色，而不是全局修改，可以直接在笔记里写 HTML。

- **输入：** `<h1 style="color:red">这是一个红色标题</h1>`
    
- **缺点：** 这种写法很笨重，违反了 Markdown 的简洁初衷，且不方便后期统一修改。**除非特殊需求，否则不建议使用。**
    

---

### ⚠️ 一针见血的建议

如果你发现设置了颜色但没生效，通常是因为：

1. **主题冲突：** 有些主题（如 Minimal）有极其复杂的 CSS 优先级，必须在 CSS 中使用 `!important`。
    
2. **输入法：** 检查 CSS 里的分号 `;` 和冒号 `:` 是否误输成了中文全角。
    
3. **模式不对：** Obsidian 分为“实时预览”和“阅读模式”，如果你只改了一个，切换模式后颜色会打回原形。我上面提供的 CSS 代码已经兼顾了两者。

---



```css
/* 设置各级标题颜色 */
.markdown-rendered h1, .cm-header-1 { color: #ff4500 !important; } /* 橙红色 */
.markdown-rendered h2, .cm-header-2 { color: #2e8b57 !important; } /* 海绿色 */
.markdown-rendered h3, .cm-header-3 { color: #1e90ff !important; } /* 闪闪蓝色 */
.markdown-rendered h4, .cm-header-4 { color: #daa520 !important; } /* 金菊色 */
.markdown-rendered h5, .cm-header-5 { color: #8a2be2 !important; } /* 紫罗兰 */
.markdown-rendered h6, .cm-header-6 { color: #5f9ea0 !important; } /* 军兰色 */

/* --- Obsidian 分色代码块（修复行内代码误伤） --- */

/* 1. 整体容器：仅针对 pre 标签 */
.markdown-rendered pre, 
.cm-content pre {
    background-color: #fafafa !important; 
    border: 1px solid #d1d1d1 !important;
    border-radius: 8px !important;
    padding: 0 !important; 
    margin: 1.5em 0 !important;
    position: relative;
    overflow: hidden !important; 
}

/* 2. 标题栏：伪元素 */
.markdown-rendered pre::before,
.cm-content pre::before {
    content: "";
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    height: 30px; 
    background-color: #ededed !important; 
    border-bottom: 1px solid #d1d1d1;
    z-index: 0;
}

/* 3. 核心修复：仅针对 pre 内部的 code，排除行内代码 */
.markdown-rendered pre > code, 
.cm-content pre > code {
    display: block !important;
    padding: 40px 16px 16px 16px !important; /* 顶部留出标题栏位置 */
    color: #24292f !important;
    font-family: 'JetBrains Mono', monospace !important;
    font-size: 13px !important;
    line-height: 1.6 !important;
    background-color: transparent !important; /* 防止行内样式干扰 */
    border: none !important;
    z-index: 1;
    position: relative;
}

/* 4. 语言标签 */
.code-block-flair {
    position: absolute;
    top: 5px !important;
    right: 12px !important;
    color: #666 !important;
    font-size: 10px !important;
    font-weight: 700 !important;
    z-index: 2 !important;
}

/* 5. 保护行内代码：确保单反引号内容正常 */
:not(pre) > code {
    background-color: #f0f0f0 !important;
    padding: 0.2em 0.4em !important;
    border-radius: 3px !important;
    font-size: 85% !important; /* 恢复正常比例 */
    color: #e57399 !important; /* 保持马卡龙粉色调 */
}

/* ================================
   强制覆盖所有加粗颜色（）
   ================================ */

.markdown-rendered strong,
.markdown-rendered b,
.cm-s-obsidian .cm-strong,
.cm-s-obsidian strong,
.cm-s-obsidian b {
    color: #A67C2E !important; /*  */
}
