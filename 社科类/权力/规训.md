在福柯（Michel Foucault）的哲学语境中，**规训（Discipline）** 并不是简单的“惩罚”或“管教”，而是一套极其精密的**社会技术**。

“规训”常用来翻译法国思想家福柯提出的概念 discipline，指一种通过纪律、训练、监视和规范化来塑造个体行为与思想的权力技术。它强调的不是简单的外部强制，而是通过学校、工厂、医院、监狱等机构，把规则内化为人的自我约束，从而形成“听话的身体”和“标准化的个体”。

它的核心目的不是消灭身体，而是\*\*“驯化”身体\*\*——把原本充满野性、混乱的人类，变成像机器零件一样“有用”且“顺从”的个体。

* * *

## 1\. 规训的三个核心手段

权力是如何在不使用暴力的情况下，让你乖乖听话的？福柯总结了三个基本招式：

**层级监视 (Hierarchical Observation)：**

权力通过“看”来控制。这不需要一个警察时刻盯着你，而是通过空间布局（如教室的讲台、敞开式的办公室、甚至街道上的摄像头）让你意识到自己**随时处于被看见的可能中**。这种“被注视感”会迫使你开始自我约束。

**规范化裁决 (Normalizing Judgment)：**

规训不关心你是否“犯罪”，它关心你是否“正常”。它建立一套细碎的标准（比如：准时、勤奋、坐姿端正、成绩达标）。如果不达标，你不会被处死，但会被视为“落后”或“不正常”，需要接受“矫正”或“辅导”。

**检查 (Examination)：**

这是监视与规范化的结合（如考试、体检、年度考评）。通过检查，每个人的情况都被转换成**数据和档案**。权力由此掌握了你的全部信息，并能根据这些数据对你进行分类、排名和管理。


* * *

## 2\. 规训的空间载体：全景敞视建筑

福柯用\*\*“全景敞视监狱”（Panopticon）\*\*来隐喻规训社会的运作。

在这种监狱里，囚犯看不见监视塔里有没有人，但他知道自己可能正被盯着。结果就是：**囚犯成了自己的监视者****。** 这种机制被推广到了整个社会：

**学校：** 用铃声、考勤、排座来管理学生。

**工厂/公司：** 用流水线、KPI、考勤打卡来管理员工。

**医院：** 用体检指标、临床分类来管理身体。


* * *

## 3\. 规训的结果：“驯服的肉体”

规训最可怕的地方在于，它不仅约束你的行为，还**重塑你的灵魂**。

**从外向内：** 最初你是因为怕罚才准时打卡，久而久之，“准时”成了你的自觉，你会为自己的迟到感到羞愧。

**生产性：** 规训不只是说“不”，它还在说“你可以”。它把你培养成一个高效的打工人、一个标准化的公民。你变得更有能力（Useful），但也变得更加顺从（Docile）。


* * *

## 总结：规训与压迫的区别

|     |     |
| --- | --- |
| **传统压迫 (Oppression)** | **现代规训 (Discipline)** |
| **显眼：** 绞刑架、铁链、暴力 | **隐蔽：** 考勤表、排名、心理测评 |
| **负面：** 剥夺你的生命或财产 | **正面：** 赋予你技能，把你变“好” |
| **断续：** 犯错时才出现 | **持续：** 渗透在每一秒的时间表里 |

**一句话理解：**

规训就是一种让你在甚至没有感到被强迫的情况下，主动把自己磨合成社会这台大机器上最合适的那颗螺丝钉的技术。
