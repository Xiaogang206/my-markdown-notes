好的，根据中国古代的科举制度，“进士”的等级和地位远高于“举人”。

简单来说，这是一个金字塔形的晋级过程：

1. **秀才 (童生考过院试)**：这是最基础的功名，获得了读书人的身份。

2. **举人 (考过乡试)**：秀才可以参加在各省省城举办的“乡试”。考中了，就叫“举人”。

3. **进士 (考过会试和殿试)**：举人才能有资格去京城参加“会试”。考中了会试，才有资格参加皇帝亲自主持的“殿试”。通过了殿试的人，才能被称为“进士”。


因此，进士是科举考试的最高功名，是“天子门生”，可以直接被授予官职。而举人虽然也有了做官的资格，但通常只能担任一些级别较低的官职或等待空缺。

* * *

### 📚 区别详解

|     |     |     |
| --- | --- | --- |
| **对比项** | **举人 (Jǔrén)** | **进士 (Jìnshì)** |
| **如何考取** | 通过在**省城**举办的“**乡试**” | 成为举人后，通过在**京城**举办的“**会试**”和皇帝主持的“**殿试**” |
| **考试难度** | 难度高，每三年一次。 | 难度极高，是科举的最高一级。 |
| **身份地位** | 获得了做官的资格，是地方上的高级知识分子。 | “天子门生”，国家的精英储备干部。 |
| **仕途前程** | 一般只能担任县一级的副职（如教谕）或等待候补，仕途空间有限。 | 立即获得官职，如进入翰林院或被派到中央各部委，起点很高，日后成为宰相、巡抚等高官的，基本都是进士出身。 |
| **第一名** | 乡试第一名称为“**解元**” | 殿试第一名称为“**状元**” (二甲第一名叫“传胪”) |
