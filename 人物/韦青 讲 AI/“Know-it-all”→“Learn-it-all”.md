这两个短语通常被放在一起讨论，用来描述两种截然不同的思维方式和职场态度。

它们之所以出名，很大程度上是因为微软现任 CEO 萨提亚·纳德拉（Satya Nadella）在企业文化转型中大力推崇从“Know-it-all”向“Learn-it-all”的转变。

* * *

### 1\. Know-it-all (无所不知者)

在英语中，这是一个带有贬义的词汇。

**字面意思：** 觉得自己什么都懂的人。

**心态特征：** 固执、自大、拒绝听取他人建议。他们认为自己的知识已经足够，不需要再改变，常表现为在会议中急于证明自己是最聪明的。

**职场表现：** 为了维持“权威”形象，往往不敢承认错误，害怕失败，这会导致团队沟通阻塞和创新停滞。


### 2\. Learn-it-all (无所不学者)

这是一个褒义且具有前瞻性的词汇。

**字面意思：** 想要学习一切的人。

**心态特征：** 谦逊、好奇、拥有**成长型思维（Growth Mindset）**。他们承认自己并不知道所有答案，但愿意通过尝试和失败去寻找答案。

**职场表现：** 善于倾听，把挑战看作学习的机会。这种人更能适应快速变化的技术环境。


* * *

### 核心对比

|     |     |     |
| --- | --- | --- |
| **维度** | **Know-it-all (无所不知)** | **Learn-it-all (无所不学)** |
| **核心动力** | 证明自己有多聪明 | 弄清楚事情是怎么回事 |
| **对待错误** | 掩饰错误，视其为耻辱 | 承认错误，视其为学费 |
| **对待反馈** | 感到被冒犯、防御心强 | 积极寻求反馈以改进 |
| **口头禅** | “我早就知道了。” | “我不确定，但我可以学。” |

### 为什么这很重要？

纳德拉曾引用斯坦福大学心理学家卡罗尔·德韦克的理论说：

“在长期竞争中，**‘无所不学者’（Learn-it-all）总是会打败‘无所不知者’（Know-it-all）**，即使后者起步时拥有更多的天赋。”

在当今 AI 和技术飞速迭代的时代，过去积累的知识（Know）可能很快过时，而持续学习的能力（Learn）才是核心竞争力。
