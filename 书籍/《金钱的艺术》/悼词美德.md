**“悼词美德”**（eulogy virtues）是一个非常有名的概念，出自美国作家、纽约时报专栏作家**戴维·布鲁克斯**（David Brooks）的书《品格之路》（The Road to Character，2015年出版）。

简单来说：

**悼词美德** = **别人在你葬礼/追悼会上，****会愿意说出口、真正怀念你的那些内在品质**。

它和另一个词形成鲜明对比：|

|     |     |     |
| --- | --- | --- |
| 对比项 | 简历美德<br>（resume virtues） | 悼词美德（eulogy virtues） |
| **定义** | 你写在简历/CV里、拿去求职/炫耀的成就和技能 | 别人在你悼词/葬礼上会提到的品格和对人的影响 |
| **谁在说** | 主要是**自己**对外展示的 | **别人**（亲友、同事）发自内心的评价 |
| **关注点** | 外在成功：职位、薪水、奖项、业绩、头衔 | 内在品质：善良、诚实、勇敢、慷慨、忠诚、温暖、有同理心、是否真正爱过别人、是否帮助过别人 |
| **例子** | “连续三年超额完成KPI”“拿到哈佛MBA”“管理过百人团队” | “他总是第一个想起朋友生日”“她在我最难过时陪我熬夜”“他从不背后说人坏话”“他让身边的人都觉得自己被看见、被珍惜” |
| **本质** | 市场价值、竞争优势、自我包装 | 灵魂深度、人际关系的真实质量、生命的意义 |

### 核心思想（布鲁克斯的原意）

现代社会让我们拼命追求“简历美德”（升职加薪、社会地位），却常常忽略“悼词美德”。 

但当生命结束时，真正留在人们记忆里的，几乎从来不是你赚了多少钱、职位有多高，而是**你是一个怎样的人、你如何对待他人**。

一句话总结最流行、最扎心的版本：

你花了一辈子精心打磨简历， 

但最后别人读的，是你的悼词。

这个概念在中文互联网（尤其是知识付费、罗振宇得到App、得到专栏、知乎、微信公众号）被广泛传播和讨论，常被用来提醒人们：**别只顾着“成功”，更要问自己——死后别人会因为什么而真正怀念我？**
