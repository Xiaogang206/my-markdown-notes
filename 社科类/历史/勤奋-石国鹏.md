来源对经济与勤奋观的讨论，批判性地审视了“勤奋”这一标签在不同社会形态下的真实含义和效用，并将其置于中国古代小农经济、现代商品经济以及专制政治制度的宏大背景之下。

### 一、 勤奋观的历史演变与经济形态的制约

来源指出，“勤奋”作为中国人在全世界的标签，主要指近一百年、一百多年以来的现象。在古代，这种标签并不准确，因为勤奋本身在特定经济形态下是无效的。

1. 小农经济下的无效勤奋


在中国古代，在小农经济占据绝对主导地位时，中国人民并不勤奋，或者说勤奋也没用。

* **靠天吃饭：** 农民的收入主要取决于\*\*“老天爷赏不赏饭吃”\*\*，而不是勤奋。
* **不产生迭代：** 很少有农民会想方设法去发展技术、革新或努力劳动。如果没有**商品经济的刺激**，生产出来的东西再多也没用。
* **生存状态：** 中国古代农民的基本特色是\*\*“半年忙、半年闲”**。绝大多数人在风调雨顺时**勉强温饱\*\*，天灾人祸时则**饿殍遍野**。这并非昏君或天灾所致，而是这种**经济形态本身就养不活那么多人**。因此，历史上所谓的治世和盛世（如康乾盛世、大唐盛世）实际上**与老百姓无关**，老百姓该饿死还是要饿死。

2. 商品经济下的勤奋驱动


直到改革开放以后，随着商品经济涌入农村，农民才变得更加勤奋。

* **环境决定论：** 城市人比农村人更勤奋，并非因为城里人更高尚，而是**城市里的经济环境就是这样决定的**。
* **富足与休息：** 来源观察到，与富足的西方、北欧、美国、澳洲等国家相比，当代中国人民付出了巨大的努力，但获得的休息时间、假期和幸福指数却**远远不如**。

### 二、 无效努力与社会形态的内卷

来源认为，当前中国人民付出的巨大努力中，有很大一部分是无效的努力，即所谓的内卷（Involution）。

* **内卷的定义：** 内卷被定义为**不产生迭代和进步的努力**。
* **进化的对立：** 内卷的反义词不是“躺平”或“摆烂”，而是**进化（Evolution）**，即**产生迭代和进步的努力**。
* **人类的底线：** 来源强调，**人类没有躺平的选项**。如果人类作为一个整体选择躺平，唯一的后果就是**全族灭亡**，因为人类的力量不足以与自然界的猛兽对抗。因此，人类享受休息和假期，是为了**积攒精力，更好地去劳动**，用以提升自我。

### 三、 专制政治制度对勤奋的抑制

在专制主义中央集权的政治制度与社会形态中，统治者的核心目标与民众的勤奋和富裕是直接矛盾的。

1. **统治基础：** 专制帝王最关心的第一件事是**如何顺畅地把皇帝之位传下去**，确保权力不被分享。
2. **贫民政策：** 专制主义中央集权导致的结果是统治者**一定要让老百姓服服帖帖地成为他的顺民**。因此，统治者会实施\*\*“贫民”\*\*（让老百姓穷）的政策，这是商鞅“驭民五术”中最重要的一点。
3. **避免闹事：** 如果老百姓富了，就会\*\*“吃饱了撑的就要闹事”\*\*。这是所有专制统治者都要强调的一点。
4. **闭关锁国与改革停滞：** 尽管清朝统治者清楚地知道，中国的土地在古代技术下**不可能养活 4 亿人**，但他们仍选择**闭关锁国**，不引进促进海外贸易和工业化的措施。他们之所以不进行改革，是因为他们担心这些改革会**危及自己的绝对权力**。闭关锁国是**专制帝王自保**的一种方式，同时也是**小农经济的体现**。

综上所述，勤奋本身不是衡量社会进步或个人幸福的最终标准。在不同的经济形态下，勤奋的有效性存在巨大差异。在古代，小农经济体制和专制政治体系共同作用，使得勤奋成为一种无效的努力，甚至会因威胁统治者的权力而被抑制。现代的努力则面临着转化为内卷的风险，教育和政治制度需要引导人们将努力导向进化，才能实现真正的进步。

* * *

可以这样理解： 经济形态决定了勤奋的效率，而政治制度决定了勤奋的上限。在古代小农经济和专制制度下，勤奋就像在一个有漏的桶里打水——效率低下（经济决定），并且统治者还会故意限制打水的工具，生怕你打得太多反过来威胁到他的水库（政治决定）。只有在商品经济和促进迭代的环境中，勤奋才能转化为进化。

# “involution 和 evolution 的异同”

### **1\. 词源对比**

**evolution**（进化）： 

源自拉丁语 _evolutio_（展开），由 _e-_（出、外）+ _volvere_（滚动）构成，字面义“滚出、展开”，后引申为“事物从简单到复杂的发展”。

**involution**（内卷）： 

源自拉丁语 _involutio_（卷入），由 _in-_（内、入）+ _volvere_（滚动）构成，字面义“卷入、缠绕”，后引申为“向内收缩、复杂化”。


### **2\. 核心含义对比**

|     |     |     |
| --- | --- | --- |
| **维度** | **evolution（进化）** | **involution（内卷）** |
| **方向** | 向外、向前的**发展性变化**（从低级到高级、简单到复杂） | 向内、向回的**收缩性变化**（无实质进步的内耗、重复） |
| **典型场景** | 生物进化（达尔文理论）、技术进步（科技进化）、社会发展 | 社会/学术内卷（如教育竞争、职场内耗）、数学/生物学的“内卷绕”（如器官折叠） |
| **结果** | 产生**新结构、新功能**（如物种演化出翅膀） | 陷入**无意义的复杂循环**（如学生刷题竞赛却未提升能力） |

### **3\. 用法差异**

**evolution** 是中性偏褒义的词，强调“进步性发展”，常见搭配：_biological evolution_（生物进化）、_technological evolution_（技术演进）。

**involution** 近年因“内卷”概念流行而广为人知，多带**负面色彩**，指“过度竞争导致的内耗”，常见搭配：_educational involution_（教育内卷）、_workplace involution_（职场内卷）。


简单来说：

**evolution** 是“向外突破式的发展”，比如从猿到人的演化；

**involution** 是“向内缠绕式的内耗”，比如同一赛道上无意义的重复竞争。
